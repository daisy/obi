Transport bar

Index of shortcuts

  + Del ................ Edit > Delete
  + F1 ................. Help > Contents
  + Shift+F1 ........... Help > View help in external browser
  + F2 ................. TOC > Rename section
  + F2 ................. Strips > Rename strip
  + F3 ................. Edit > Find next
  + Shift+F3 ........... Edit > Find previous
  + Alt+F4 ............. File > Exit

  + F6 ................. View > Focus on TOC view
  + F7 ................. View > Focus on strips view
+ F8 ... Focus on transport bar if enabled
  + Ctrl+Shift+Left .... TOC > Move section out
  + Ctrl+Shift+Right ... TOC > Move section in
  + Ctrl+Space ......... Transport > Play > Play all
  + Ctrl+Shift+Space ... Transport > Play > Play selection / pause
  + Space .............. Transport bar > play / pause in strip view
+ Alt + Right ... play next phrase
+ Alt + left ... play from begining of phrase / play previous phrase
+ Alt+Down .... Play next section.
+ Alt+Up .... play previous section.
+ Control+comma ... Rewind.
+ control+period ....FastForward.

  + Ctrl+A ............. TOC > Add section  :Add strip  -:Avn - highly conflicting shortcut, marked for change
  + Ctrl+A ............. Strips >  :Add strip  -: Avn - highly conflicting shortcut, marked for change
  + Ctrl+Alt+A ......... TOC > Insert section
  + Ctrl+Alt+A ......... Strips > Insert strip
  + Ctrl+Shift+A ....... TOC > Add sub-section
  + Ctrl+B ............. Blocks > Add empty block
  + Ctrl+Shift+B ....... Blocks > Add empty page blocks

  + Ctrl+G ............. FREE
  + Ctrl+I ............. Blocks > Import audio files
  + Ctrl+J ............. FREE
  + Ctrl+K ............. Block > Block is used
  + Ctrl+K ............. Strips > Strip is used
  + Ctrl+K ............. TOC > Section is used

  + Ctrl+Shift+L ....... Edit > Delete unused data
  + Ctrl+M ............. Blocks > Merge block with next

  + Ctrl+Shift+M ....... Strips > Merge with next strip
  + Ctrl+N ............. File > New project
  + Ctrl+Shift+N ....... File > New project from import
  + Ctrl+O ............. File > Open
