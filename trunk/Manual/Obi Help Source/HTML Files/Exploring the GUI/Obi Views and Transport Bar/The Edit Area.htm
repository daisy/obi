The Edit Area

 

When you select an entry in the list, its name and content appear in the edit area of the Metadata view. You can modify the content of the entry here.

 

The edit area has the following fields:

More options

Button

 

Provides the list of advanced options to update metadata. The details of these options are given below.

 

  Note: There are some entries that cannot be modified. The edit boxes for such entries are set to read-only.

 

Advanced Options for Metadata Updation

You can save the metadata of the current project as default metadata, which can then be used across projects. This feature enables users to fill the metadata entries once and reuse it later by simply a click of a button. The More Options button in the edit area of Metadata view gives you access to this functionality via the following commands:

        Insert default metadata: Fill the empty metadata entries using the default metadata.

        Overwrite with default metadata: Set the metadata entries to the values stored in default metadata. Please note that this command will replace all the metadata entries.

        Save as default metadata: Save the metadata of the current project as default metadata.

 

Please note the following:

        The metadata entries dc:Identifier and dc:Date are not stored in the default metadata.

        Make sure that only the entries that can be duplicated across the projects are stored in the default metadata.
