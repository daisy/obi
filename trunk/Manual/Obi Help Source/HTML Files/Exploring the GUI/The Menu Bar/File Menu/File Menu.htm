File menu

 

Enables you to perform basic file operations like creating a new Obi project, opening an existing one, saving, and closing it.

 

The figure below illustrates how the file menu looks:

Figure: File Menu

 

File menu lists the following items:

Menu Item

Keyboard Shortcut

New project

Ctrl+N

New project from import

Ctrl+Shift+N

Open project

Ctrl+O

Restore from backup

-

Close project

Ctrl+W

Save project

Ctrl+S

Save project as…

Ctrl+Shift+S

Recent project

Has a submenu

Exit

Alt+F4
