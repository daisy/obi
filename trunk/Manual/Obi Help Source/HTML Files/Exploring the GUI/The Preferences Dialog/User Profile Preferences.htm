User Profile Preferences

 

The User Profile Preferences tab shows user’s information extracted from your operating system. However, you can change it. The figure below shows how the User Profile Preferences tab looks:

Figure: User Profile Preferences

The preferences dialog offers the following options under this tab:

Field

Field Type

Description

Full Name

Edit Box

Full Name of the user

Organization

Edit Box

Name of the organization

Language

Combo Box

Language currently used by Obi. You may set a different language for working in Obi by selecting it from the list. Please note that this change will have effect only when you restart the application.

 

Remarks:

In case of a language switch, Obi reconfigures the GUI only if the language opted in the dialog is supported by it. But in either of the cases, it updates the metadata entry dc:Language.
