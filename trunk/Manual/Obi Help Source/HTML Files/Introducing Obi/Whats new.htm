What’s new?

 

Obi 2.0 presents a rich new feature set with most awaited features like DAISY book import and configurable keyboard shortcuts. Other advancements include recording toolbar for book narrator, advanced recording mode, project recovery among many others. It also incorporates enhancements to existing operations of Obi along with regular bug fixes.

 

The introductions and improvements of this release are listed below:

DAISY book import

An existing DAISY book is now modifiable using Obi. Simply create a project by importing an existing DAISY book. Obi will import the structure and audio from the book in the project just created. Thereafter, you can work on it using Obi’s usual operations.

Please note that in the process of DAISY book import, if some errors occur due to discrepancies in the duration of audio content of a phrase and its entry in the SMIL file, they can be eliminated by adjusting the Import tolerance in the Project preferences tab of the Preferences dialog.

 

 

New additions to Graphical User Interface for book narrators

Recording toolbar is a new control bar to smoothen the process of recording and structure mark-up while narrating the contents from user’s computer screen. The small toolbar rests on one side of the screen, providing maximum space for the window from which the narrator has to read out.

 

Enhanced export operation

The function Export as DAISY contains the following enhancements and adjustments:

  • Integrates MP3 converter: Users can choose to have exported audio files directly in MP3 format, eliminating the need of an additional step.
  • Allows export of blank pages: Unlike other blank phrases, Obi will now export blank page phrases for DAISY 2.02 export.
  • DAISY 2.02 workaround for page phrases: Shifts the page phrases existing at the foremost position in the section to make exported DTB compliant with DAISY 2.02 specifications.

 

Configurable keyboard shortcuts

Obi now enables its users to alter the shortcut key combination for Obi’s operations. The Keyboard shortcut tab is augmented to the Preferences dialog, which displays the list of Obi’s operations and their corresponding shortcut keys. Users can modify the keyboard shortcut of an operation by selecting it in the list and assigning new shortcut key combination.

 

Project Recovery

The process of restoration of project from the backup file has been automated. The command Restore from backup, under the File menu on the menu bar, reopens the project using the backup file. You can, then, choose to instate the backup file in place of the project file or reinstate the original project after administering the contents of the project and their validity.

 

Advanced recording mode

An advanced recording mode has been introduced with the following behavior:

  • Start recording from cursor erasing the following audio
  • Always record directly from transport bar
  • Allow overwrite option

Advanced recording mode can be enabled or disabled using Enable/Disable advanced recording mode button in the Audio preferences tab of the Preferences dialog.

 

Delimit Content view width

The width of Content view can be fixed or made variable using the checkbox Fix Content view width in the project preferences tab of the Preferences dialog. If you choose to fix the width of Content view, the orientation of other visible phrases will not be disturbed if the length of one of the phrase exceeds the width of Content view. Otherwise, the width of Content view will be equated to the length of the elongated phrase and other phrases will be adjusted accordingly.

Bookmark

This release of Obi introduces bookmark, which enables users to save the position where they are currently working and directly reach there when the project is reopened. Any section or phrase can be set as bookmark and the commands pertaining to bookmark can be found in the Edit menu.

 

Export audio for external processing

Export audio is a new operation that lets users to export audio contained in a phrase or section to an external file. This feature is very useful when users have to perform audio processing using some other tool.

 

Jump to a location using time

Users can now jump to any location in the audio content of project. The Go to… dialog has been enhanced to find a location in the audio content of an item using time.

 

Extended Merge sections function

The Merge sections function has been extended to merge multiple sections. You can merge any number of contiguous sections using this variation of command.

 

Retention of initial silence for manual inspection

During the auto-split operation, Obi can retain the initial silence portion of audio contained in the phrase. This feature would avoid deletion of some useful content that may have amplitude lower than the silence threshold. After the split, the user may examine the contents of the retained initial silence for any useful portions and do the needful.

 

Adjustable duration of audio operations

The duration of the following commands can be modified using the fields Operation and Duration in milliseconds in the Audio preferences tab of the Preferences dialog:

  • Nudge
  • Preview
  • Play with elapse back

 

Enhanced Time display control

The Time display control of Transport bar has new set of options to make it more informative and utilizable
