Frequently Asked Questions

 

Q. Can I open multiple instances of Obi?

Yes, you can open as many instances as you wish. See the topic Starting Obi for starting an instance.

 

Q. Can I open multiple projects in an instance of Obi?

No, you can only open a single project. But you can open multiple instances of Obi and open a different project in each of them.

 

Q. Is the project file the same as the DAISY DTB?

No, the project is a xuk file, whereas the DAISY DTB is a collection of audio, navigation and synchronization files. A DTB is created when you export the project as DAISY.

 

Q. I do not have a daisy player, how can I hear the DTB?

You can sort all the audio files (with .wav/.mp3 extension) in alphabetic order and play them one by one in any player that supports wav/mp3 format.

 

Q. What is a unique ID?

Various schemes are available for identifying digital publications. In the DTB domain, the requirements for an identifier are simply to identify the publication in a manner that is highly likely to be globally unique. A major purpose of the uniqueness requirement is to prevent filename collisions among bookmark files.

To meet this base requirement, a simple DTB id scheme might be used. A DTB identifier under this scheme consists of a hyphen-separated string consisting of a two-letter country code drawn from [ISO 3166], an agency code unique within its country, and an identifier unique within the agency. For example, us-afb-x12345.

 

Q. Can I select multiple items in Obi?

You cannot select multiple items arbitrarily. But, when you select a section, all the phrases get selected, and if you select a section in TOC View, the subsections and their phrases get selected, too.

 

Q. Can I copy an item from one Obi project to another?

It is not possible to use the copy command across several instances of Obi. But, you can import audio files from the data folder of one project to another.

 

Q. What audio formats can be imported in Obi?

You can import audio files of wav/rif format (having a .wav extension). Please note that the audio files should have the same audio settings (like channel and sampling rate) as your project.

 

Q. Can I import a DAISY DTB in Obi?

No, you cannot import a DAISY DTB in this version, but a future version may support this feature.

 

Q. I have cleaned unreferenced audio, can I revert to previous state? What can I undo?

You cannot revert to an earlier state, once you have executed this command as it deletes all the audio data from the disk. However, the audio that is referenced in the undo record remains and can be reverted, if you perform redo operation(s).

 

Q. If I have not saved the project, and a crash occurs, what can I expect to loose?

All the data entered after you last saved the project will be lost. The audio data however, would remain on the disk till you re-open the project in Obi, but it would not be referenced in the project file.

 

Q. Can I hide the Content View?

No, the Content View cannot be hidden.

 

Q. When can I use blank phrases?

Blank phrases act as placeholders for audio content. You may use them for leaving a space for some audio content while recording. These can also be useful while creating a skeleton of the book.

 

Q. What if I do not mark any phrase of a section as a heading phrase?

If you do not assign a heading role to any phrase of the section explicitly, the first phrase would be considered the heading phrase.

 

Q. If the heading phrase is not the first phrase of the section, would the sequence of playing the phrases of the section change?

Obi will play all the phrases of the sections one by one from the first to the last. So, the audio from the heading phrase will be played after the audio from the preceding phrases have been played.

 

Q. Why are silence phrases marked as Unused?

Silence phrases are marked as Unused as they are used only for editing and should not appear in the DTB.

  

Q. How do I check for duplication of page numbers?

Duplication of page numbers is checked automatically before exporting, when you give the Export as DAISY command. But, you can also check manually if you wish to do so.

 

Q. How can I check for the correctness of the sequence of page numbers?

You can only check the sequence manually.

 

Q. Is there any auto correct option for page numbers?

No, Obi will point out the duplicate page numbers while exporting the project as a DAISY DTB, but you have to correct them manually.

 

Q. Can I select audio from multiple phrases?

No, arbitrary multiple selection is not allowed in Obi, and the same is true for the audio contained in phrases.

 

Q. What types of books can be produced in Obi?

Obi can produce DAISY/NISO 2005 "NCX-Only" DTBs natively.  

 

Q. How can I change my microphone volume?

Your microphone volume can be changed via the system settings. Follow the steps given below to change it:

·        Click Programs->Accessories->Entertainment->Volume Control on the Start Menu.

·        Click Advanced Controls on the Options menu.

·        Move the mic in volume slider up and down for increasing and decreasing the volumes respectively.

 

Q. What is preceding silence audio?

It is a phrase that holds a bit of audio for silence in your recording environment as it may differ from place to place. A preceding silence phrase is used for silence detection while auto-splitting a phrase.

 

Q. How do I apply phrase detection with different set of values?

When you choose to apply phrase detection, a dialog box opens showing the parameters determined from the preceding silence phrase. If there is no silence phrase preceding the current one, the dialog box shows the default parameters. You can change them if you want to use different ones.

 

Q. In which formats can Obi projects be saved using “Save Project As”?

The project can only be saved as a copy of the current project. To know what and how to save, see Saving project in different location.

 

Q. What is the difference between the commands Delete Unused Data and Clean Unreferenced Audio?

Delete unused data deletes all the data items (sections or phrases) that are marked as Unused from the project, and this command can be undone. Whereas, Clean Unreferenced Audio removes the audio data that is not referenced in the project from the disk, and this command cannot be undone.

 

Q. What is the difference between zoom and audio zoom?

Zoom magnifies or shrinks everything that is visible in the work area of the Obi screen, whereas audio zoom scales the waveforms corresponding to all the visible phrases of the section horizontally.

 

Q. Why doesn’t the audio play on navigating?

The option Play on navigate may be turned off. To turn this option on click Transport->Play on navigate on the menu.

 

Q. What is play with elapse back?

If you are playing a piece of audio at a faster rate and want to hear something just heard again at a normal speed, press E. Obi will go back a certain unit of time and start playing at the normal speed.

 

Q. What is New project from import and how is it different from New Project?

New project from import allows you to import the structure of the book from an xhtml file, whereas New project would create a blank project only and you have to create the structure manually.

 

Q. How can I clear the list of recent projects?

The submenu of File->Recent Projects has a Clear List option at the end of the list of recently opened projects. Click on it to clear the list.

 

Q. How can I paste a piece of audio inside a phrase?

First, play the audio of the phrase, and then pause at the position where you want to insert the audio. Then, use the paste command to paste the audio from the clipboard. For more information, see Editing the audio.

 

Q. Are there any accessibility features for people with low vision in Obi?

Obi provides zoom and audio zoom for magnifying the elements of the Obi screen. To magnify or shrink all the elements of the Obi screen, press Ctrl+Alt++ or Ctrl+Alt+- respectively. To stretch or skew the waveform corresponding to a particular phrase, press Ctrl+Alt+Shift++ or Ctrl+Alt+Shift+- respectively. To further assist you while working, audio clues are provided.

 

Q. Can I mark or select the phrase for editing at a later stage during the time of recording?

Yes, you can do it by pressing F9. The current phrase would be marked as Todo, and Obi would create a new one next to it in the same section for recording. For more information on marking the structure while recording, see Create the Structure while Recording.

 

Q. How can I zoom the phrase? Can I zoom a particular phrase only?

You can zoom the waveforms corresponding to all the visible phrases of a section by selecting Audio Zoom in or Audio zoom out from the View menu.

  

Q. How can I convert DTB audio to MP3? Can I convert it into any other format?

You can encode an exported DTB to MP3 by selecting Z3986 DTB Encoder from the Tools Menu. This version of Obi does not allow you to encode in any other format.

 

Q. Can I make out the elapsed or remaining time during playback?

Yes, the Transport Bar has a Time Display control that shows the elapsed/remaining time during playback. You can also hear the time by pressing F8.

 

Q. I encountered a package error code 2896 while installing Obi on Windows Vista. What should I do?

You must have the administrator privileges for installing Obi. Follow the instructions given below to install Obi on your system:

·        Run the Command Prompt with administrator privileges.

·        Go to the directory where the setup file is stored.

·        Type setup.msi on the Command Prompt and press Enter.

The installer will start installing Obi.
