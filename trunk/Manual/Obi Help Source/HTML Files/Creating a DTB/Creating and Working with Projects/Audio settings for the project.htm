Audio Settings for the Project

 

After creating a new project in Obi, it is essential to give the information regarding the audio format in which the project’s data will be stored. This information should be entered carefully as it can not be changed afterwards. Obi supplies a separate dialog box for this purpose, which gets displayed automatically during the project creation process.

Figure: Audio settings for the project dialog

It has the following fields:

Field

Field Type

Description

Sample rate

Combo box

Sampling rate to be used for the audio content of the project. Please note that higher values will result in better audio quality but consume more disk space.

Audio channel

Combo box

Type of channel used for the project’s audio. Mono channel will produce the same sound effect in all the speakers, whereas the stereo sound will vary in the left and right speakers. It is noted that mono channel is best for speech or lectures, whereas stereo channels produce natural effects for theatre and stage performances.

 

 After supplying relevant values, click on the OK button, and the interrupted command (New project or New project from import) will resume execution.
