Assigning a Page Role

 

You can assign a page role to any phrase by clicking Phrases->Assign Role-> Page on the menu or by pressing Ctrl+Shift+G.

 

A dialog box appears as shown in the figure below:

Figure: Set Page Number

It has the following fields:

Field

Field type

Description

Page number

Edit Box

Enter the page number you wish to give to the current page. Obi suggests a page number in this field, 1 more than the previous one.

Number of Pages

Edit Box

Not applicable for this operation.

Page Kind

Combo Box

Select the page kind: page-front, page-normal, page-special.

Renumber following pages

Check Box

If checked, Obi increments the page number of subsequent pages by 1. For example: If we set the first page number as 2, the next page will have the page number 3, then, the next one will have 4 and so on.

 

Note that when you choose to renumber the pages, the pages marked as page-special are ignored, as special pages can have alphanumeric page numbers, such as “A34”.
