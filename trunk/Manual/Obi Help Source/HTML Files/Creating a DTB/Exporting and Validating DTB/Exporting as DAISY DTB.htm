Exporting the Project

 

Obis export functionality lets you export the project in the following formats:

        DAISY 3.0 audio-ncx

        DAISY 2.02 audio-ncc

        EPUB 3.0 audio-only with navigation

 

Follow the steps given below to perform the export operation:

  1. Click Tools->Export as DAISY/EPUB on the menu, or press Ctrl+E.
  2. A dialog box appears that allows you to select the specifications to which the exported publication should comply.
  3. Select the appropriate radio button and click on the OK button. The export dialog opens as shown in the figure below:

Figure: Export dialog (in case of EPUB Export)

 

  1. Obi suggests a path for the export directory in the dialog box. But you can change it if you want to store the exported DTB somewhere else. You can also browse for a location by clicking the Select button.
  2. While exporting, Obi creates one audio file per section. But you can restrict this behavior to a certain level (depth of section) by choosing an appropriate value from the combo box named Create one audio file per section till. All the sections after that level will be clubbed into one file. For example, if you choose level 4 in the combo box, Obi will create one file per section till level 4. The subsections of the section at level 4 will be appended in the same file as the section at level 4. A new file will only be created when the section with level 4 or less is encountered.
  3. If you want to have audio files in MP3 format in your exported DTB, check the checkbox Encode to MP3. In this case, you must select a suitable bit rate for MP3 files; the higher values ensure better quality but consume more disk space. If this checkbox is unchecked, the exported audio files will be in WAV format. Please note that for EPUB export, it is checked by default as EPUB specifications do not support WAV files. When this check box is checked, Obi activates the Advanced button that provides access to the following advanced formatting options for the exported MP3 files:

a)      Resample at 44.1 KHz

b)      Replay Gain

c)      Stereo-type

  1. You can also instruct Obi to add section names to audio file names by checking the checkbox Append section name to audio file name. In this case, the exported audio file will include the name of the contained section at the end.
  2. If the above checkbox is checked, the Filename length limit checkbox will get activated. If you want to restrict the number of characters in exported file name, check the checkbox and enter the max length in the numeric edit box thus enabled.
  3. For EPUB export, you can specify the maximum length for EPUB file. Check the checkbox EPUB Filename Length and enter the maximum length in the adjacent numeric edit box.
  4. Click on the OK button to start exporting.

 

Obi creates a subdirectory of your project directory and exports the files into it. The files will be exported according to the standard (DAISY 3/DAISY 2.02/EPUB 3) chosen before exporting. For example, if you chose DAISY 2.02, the files will conform to DAISY 2.02 specifications.

 

Please note that while exporting according to DAISY 2.02 specifications, Obi will shift any page phrase appearing at the beginning of a section to the last position of the prior section.

 

Checks performed before exporting

Before exporting, Obi checks for the following:

  • Duplication of page numbers for each kind of page: Obi allows you to have the same page numbers across different kinds of pages, but not within a page kind. For example, you can have page number 4 in page front, as well as in page normal. But if page number 4 is encountered twice in page normal, a message will be displayed with a warning.
  • Empty sections or phrases: Obi does not export empty sections or phrases to the DTB. But during DAISY 2.02 export, Obi will permit blank page phrases to exist in exported DTB. This is done purposefully to provide flexibility to the publisher in the scenario where the indented audience does not wish to hear page numbers while listening to audio of the book.
  • Empty anchors or skippable notes: Obi will also check for empty anchors or skippable notes in the project, and display a warning message if it finds them.

 

Some important files exported by Obi

The following table lists the important files exported by DAISY export.

Description

The navigation control file enabling the users to move smoothly among files while synchronization.

 In case of the EPUB export, Obi will compress together the audio, text, navigation, and synchronization files in a single file. This file will bear .epub extension and contain the publication in entirety.

 

You can play the exported book using a DAISY/EPUB player, which can be downloaded from the following links:

        DAISY: http://daisy.org/tools/tools.shtml?Cat=playback

        EPUB 3: http://readium.org/

 

You can also hear the contents of the book in any WAV/MP3 player. Each audio file contains one or more sections of the DTB and the audio files are named with a number suffix. If you play these files in an increasing order of their number suffix, the book will be played in the correct order. However, you will not be able to take advantage of navigation, a salient feature of DAISY/EPUB book.
