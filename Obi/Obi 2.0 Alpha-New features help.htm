How to use new features of Obi 2.0 Alpha

 

DAISY 3 Import

In addition to importing structure from an XHTML file, the import function has been enhanced to incorporate import of the following DAISY 3 book formats:

1)    Complete audio and structure from DAISY 3 book (Audio ncx)

2)    Audio and structure from full text full audio DAISY 3 book – The text not required by Obi, i.e. text not relating to the sections in the book, will be removed.

3)    Structure from DTBook.xml

 

Configurable keyboard shortcuts

Users can now configure the keyboard shortcuts to increase usability of Obi. The Preferences dialog offers a new tab/Page that enables users to set shortcut keys for Content view, TOC view or entire application. The Keyboard shortcut tab of the Preferences dialog has following fields:

1)    A combo box to select the scope of the keyboard shortcut i.e. Content/TOC view or menu item shortcut keys.

2)    A list view that displays the operations available in the selected scope and their corresponding shortcuts.

3)    An edit box that lets users to type the preferred shortcut key combination.

After entering the key combination to an operation, click the Assign button to replace the keyboard shortcut. You may, however, restore the defaults anytime by pressing the Restore defaults button.

 

Recording toolbar

Obi now comes with a pop-up recording toolbar that enables the users to record the book contents even when Obi is minimized. If the narrator is reading the book contents from the scanned images, the whole exercise would become very intricate, as he/she might have to adjust the screen many times. The new pop-up recording toolbar will empower the users to record the contents very easily by having a tiny bar visible on the screen and performing all the functions using it.

 

Multiple section merge

Enhanced Obi allows the users to merge two or more contiguous sections by a single command. Click Sections->Merge sections->Merge multiple sections on the menu. A dialog box appears that lets the users to select multiple sections and merge them into one. The contents of all the selected sections will be appended to the first section of the selection in the same order as before.

 

Export audio contained in a phrase or section to an external file

Now, the users can choose to export the audio contained in the selected phrase or section to an external audio file by clicking Tools->Export audio on the menu bar. The audio of the selected item will be exported to an audio file with the same audio format as audio preferences of the project. This file can be used for external processing and can be found in the directory named Exported_audio_file in the project directory.

 

Export DTB in MP3 format

The export function now allows the users to export in MP3 format directly, without requiring an extra step of encoding DTB audio to MP3. The Choose export directory dialog has an additional checkbox Encode to MP3. If this checkbox is checked, the exported audio files will be in MP3 format; otherwise they will be in WAV format.

 

DAISY 2.02 workaround for page phrases

DAISY 2.02 does not allow a page phrase to be the first page of the section, whereas the DAISY 3 standard permits it. Obi has been improved to make this adjustment automatically. The export function of Obi will move the first phrase of the current section to the last position in the prior section, thereby maintaining the order of phrases and adhering to the DAISY 2.02 specifications.

 

Blank page export

New Obi provides the users with an option to export page phrases without audio also. This feature is especially helpful in case if the book listeners do not want to hear the page numbers while hearing the audio of the book.

 

Bookmark node

Users can instruct Obi to remember the selected node even after the project is closed. The project preferences tab of the Preferences dialog has a checkbox to turn this option off or on. If the checkbox Open bookmark node when project reopens is checked, Obi will remember the selected node and place the focus there when the project is re-opened.

 

Retain initial silence

Users can instruct Obi to retain initial silence for the command auto-splitting using phrase detection. The audio preferences tab of the Preferences dialog has a new checkbox labeled Retain initial silence. If this checkbox is checked, the silence present at the beginning of the selected phrase will be retained as the first phrase in the set of phrases resulting from the process of phrase detection. If it is not checked, the initial silence will be removed.
