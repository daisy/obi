  File Menu				ALT+F

  Strips Menu				ALT+S
  Tools Menu				ALT+T
  Help Menu				ALT+H

  New Project				CTRL+N
  Open Project				CTRL+O
  Save Project				CTRL+S
  Save Project as			CTRL+SHIFT+S
  Close Project				CTRL+W
  Exit					ALT+F4

  Undo					CTRL+Z
  Redo					CTRL+Y
  Cut					CTRL+X
  Copy 					CTRL+C
  Paste					CTRL+V
  Delete				Del
  Touch Project         CTRL+T (we should keep it in the final version, this is useful.) 
   
  Hide TOC View				ALT+T
  Show TOC View				ALT+T
  
  Add section				CTRL+A
  Add subsection			CTRL+B
  Delete section			DEL
  Rename section			F2
  Move Section Up			CTRL+SHIFT+UP (otherwise this conflicts with blocks.)
  Move Section Down			CTRL+SHIFT+DOWN
  Move Section In			CTRL+SHIFT+RIGHT
  Move Section Out			CTRL+SHIFT+LEFT

  Show in Strip View			ALT+V
  Show in TOC View			ALT+V
  
  Add strip				CTRL+A
  Rename strip				SHIFT+F2 (because of selection ambivalence in strips)
  Delete strip				SHIFT+Del
  Move strip up				CTRL+SHIFT+Up
  Move strip down			CTRL+SHIFT+Down
    
  Import audio file    			CTRL+I
  Play audio block			ALT+Space (should be space alone)
  Split audio block			CTRL+P
  Merge with next audio block		CTRL+M
  Delete audio block			DEL
  Rename audio block			F2
  Move audio block forward		CTRL+RIGHT
  Move audio block backward		CTRL+LEFT
      
  Open Record Dialog			ALT+R
  
  Open Help				F1
  
  Open Metadata dialog			ALT+M
  
  Open Audio Preferences		ALT+A
   
  Open Project Preferences		ALT+P
  
  Open User Profile Dialog		ALT+U
