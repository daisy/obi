In Advance mode following checkboxes are checked:-

Audio Tab:-

Retain initial silence while phrase detection Retains the initial silence during the process of Auto-splitting using phrase detection.
Allow overwrite while recording Obi will break the selected phrase, at a position determined using a set of rules, and record in a new phrase between the two fragments.
Always record directly from transport bar Obi will deactivate the toggling behavior of Start Monitoring/Record button of Transport Bar and always show Record Directly in its place. As a result, each click on that button will initiate recording.
Limit max phrase duration to 50 minutes Sets an upper limit for the length of phrase to be 50 minutes. Otherwise, allows phrases of any length to exist.
Show live waveform while recording Draws the waveform of the audio content being recorded live.
Perform phrase detection while recording Obi applies phrase detection to the audio being recorded on the run. After the record command is stopped, the recorded content will be split at the detected positions.
Renumber the following pages when recording ends Obi renumbers the pages after the recorded audio when the record command is stopped.
Merge first 2 phrases detected by phrase detection Merges the first two phrases resulting from the process of auto-splitting using phrase detection.
Fast play without pitch change Obi preserves the pitch while increasing or decreasing the fast play rate.
Enforce single audio cursor While playing if a user pauses and selects another part of waveform in the same phrase then red cursor disappears
Prevent on the fly phrase detection on pages If checked, automatic phrase detection on pages will not happen when recording is stopped.

Project Tab:-

Fix content view width Obi will not disturb the orientation of other visible phrases even if the length of a phrase exceeds the current width of Content view. If unchecked, Obi will alter the width of Content view with each elongated phrase and shift other phrases according to the altered width.
Optimize memory utilization Optimizes memory utilized by Obi.
Enable free disk space check Checks for available disk space and warns the users during Record, Save, and Import operations.
Save project when recording ends Obi saves the modifications to the project file as soon as recording is stopped or paused.
Check for Updates Obi automatically checks for new updates to the application and informs the user.
Show waveforms in Content view Obi displays the waveforms beneath the phrase labels in the Content view. If unchecked, the waveforms are hidden and the length of the contained audio is displayed instead.
Timeout EPUB check process after 3 mins Obi terminates EPUB Check Validator in 3 minutes, which is ample time for validating even a large book. This ensures that the process does not hang up in case of an error.
Enable Mouse wheel scrolling Lets you enable or disable mouse wheel scrolling for Content view.
Increase phrase size for higher resolution If checked, phrases become larger for higher resolution.

Highlight empty sections in TOC view Obi displays the empty sections in pink background to distinguish them from others.
