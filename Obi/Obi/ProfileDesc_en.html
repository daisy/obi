In Advance mode following checkboxes are checked:-

Audio Clues If checked, enables the audio clues, disables otherwise. Audio clues provide audio indications for non-visual users.
Retain initial silence while phrase detection If checked, retains the initial silence during the process of Auto-splitting using phrase detection. If unchecked, discards the initial silence.
Preview before recording If checked, plays a small portion of audio prior to the marked cursor position before initiating the Record command. If unchecked, starts recording directly. Please note that this feature only works in the advanced recording mode.
Use Space key to stop recording If checked, Obi stops recording when user presses the Space key on the keyboard. If this check box is not checked, Obi acts according to the default behavior and pauses recording
Allow overwrite while recording If checked, Obi will break the selected phrase, at a position determined using a set of rules, and record in a new phrase between the two fragments. If unchecked, Obi will always create a phrase after the selected phrase
Always record directly from transport bar If checked, Obi will deactivate the toggling behavior of Start Monitoring/Record button of Transport Bar and always show Record Directly in its place. As a result, each click on that button will initiate recording.
Limit max phrase duration to 50 minutes If checked, sets an upper limit for the length of phrase to be 50 minutes. Otherwise, allows phrases of any length to exist.
Show live waveform while recording If checked, draws the waveform of the audio content being recorded live.
Perform phrase detection while recording If checked, Obi applies phrase detection to the audio being recorded on the run. After the record command is stopped, the recorded content will be split at the detected positions.
Renumber the following pages when recording ends If checked, Obi renumbers the pages after the recorded audio when the record command is stopped.
Merge first 2 phrases detected by phrase detection If checked, merges the first two phrases resulting from the process of auto-splitting using phrase detection.
Fast play without pitch change If checked, Obi preserves the pitch while increasing or decreasing the fast play rate.
Use Record Button to record over subsequent audio If checked,Obi starts recording over subsequent phrases on pressing record button.Also you need to check "Always record directly from transport bar" in conjunction with this checkbox
Enforce single audio cursor If checked,While playing if a user pauses and selects another part of waveform in the same phrase then red cursor disappears
After recording delete the following phrases If checked,Obi deletes following phrases when recording stops.
