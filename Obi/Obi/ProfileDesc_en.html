In Advance mode following checkboxes are checked:-

Audio Tab:-

Retain initial silence while phrase detection If checked, retains the initial silence during the process of Auto-splitting using phrase detection. If unchecked, discards the initial silence.
Allow overwrite while recording If checked, Obi will break the selected phrase, at a position determined using a set of rules, and record in a new phrase between the two fragments. If unchecked, Obi will always create a phrase after the selected phrase.
Always record directly from transport bar If checked, Obi will deactivate the toggling behavior of Start Monitoring/Record button of Transport Bar and always show Record Directly in its place. As a result, each click on that button will initiate recording.
Limit max phrase duration to 50 minutes If checked, sets an upper limit for the length of phrase to be 50 minutes. Otherwise, allows phrases of any length to exist.
Show live waveform while recording If checked, draws the waveform of the audio content being recorded concurrently.
Perform phrase detection while recording If checked, Obi applies phrase detection to the audio being recorded on the run. At the end of recording, the recorded content will be split at the detected points.
Renumber the following pages when recording ends If checked, Obi renumbers the pages after the recorded audio when the record command is stopped.
Merge first 2 phrases detected by phrase detection If checked, merges the first two phrases resulting from the process of auto-splitting using phrase detection.
Fast play without pitch change If checked, Obi preserves the pitch while increasing or decreasing the fast play rate.
Enforce single audio cursor By checking this checkbox, you instruct Obi to use only one cursor instead of two. In this case, red cursor will automatically be mapped to the location of blue.
Prevent on the fly phrase detection on pages If checked, Obi will not apply phrase detection on the page phrases while live recording.

Project Tab:-

Fix content view width If checked, Obi will not disturb the orientation of other visible phrases even if the length of a phrase exceeds the current width of Content view. If unchecked, Obi will alter the width of Content view with each elongated phrase and shift other phrases according to the altered width.
Optimize memory utilization If checked, optimizes the memory utilized by Obi.
Disable roll back for cleanup If checked, Obi will not allow the users to rollback the Cleanup command.
Enable free disk space check If checked, checks for available disk space and warns the users during Record, Save, and Import operations.
Save project when recording ends If checked, Obi saves the modifications to the project file as soon as recording is stopped or paused.
Check for Updates If checked, Obi automatically checks for new updates to the application and informs the user.
Show waveforms in Content view If checked, Obi displays the waveforms beneath the phrase labels in the Content view. If unchecked, the waveforms are hidden and the length of the contained audio is displayed instead.
Timeout EPUB check process after 3 mins If checked, Obi terminates EPUB Check Validator in 3 minutes, which is ample time for validating even a large book. This ensures that the process doesn’t hang up in case of an error.
Enable Mouse wheel scrolling Lets you enable or disable mouse wheel scrolling for Content view.

This is your custom profile.Following are the descriptions of the checkboxes that may be checked depending upon your customization:-

Audio Tab:-

Audio Clues Enables the audio clues, disables otherwise. Audio clues provide audio indications for non-visual users.
Retain initial silence while phrase detection If checked, retains the initial silence during the process of Auto-splitting using phrase detection. If unchecked, discards the initial silence.
Use record button to preview before recording If checked, plays a small portion of audio prior to the marked cursor position before initiating the Record command. If unchecked, starts recording directly.
Use Space key to stop recording If checked, Obi stops recording when user presses the Space key on the keyboard. If this check box is not checked, Obi acts according to the default behavior and pauses recording.
Allow overwrite while recording If checked, Obi will break the selected phrase, at a position determined using a set of rules, and record in a new phrase between the two fragments. If unchecked, Obi will always create a phrase after the selected phrase.
Always record directly from transport bar If checked, Obi will deactivate the toggling behavior of Start Monitoring/Record button of Transport Bar and always show Record Directly in its place. As a result, each click on that button will initiate recording.

Perform phrase detection while recording If checked, Obi applies phrase detection to the audio being recorded on the run. At the end of recording, the recorded content will be split at the detected points.
Renumber the following pages when recording ends If checked, Obi renumbers the pages after the recorded audio when the record command is stopped.
Merge first 2 phrases detected by phrase detection If checked, merges the first two phrases resulting from the process of auto-splitting using phrase detection.
Fast play without pitch change If checked, Obi preserves the pitch while increasing or decreasing the fast play rate.
Use Record Button to record over subsequent audio If checked, pressing the Record button will result in replacing the audio content subsequent to the current position by the audio being recorded.Also you need to check "Always record directly from transport bar" in conjunction with this checkbox.
Enforce single audio cursor By checking this checkbox, you instruct Obi to use only one cursor instead of two. In this case, red cursor will automatically be mapped to the location of blue.
After recording delete subsequent phrases If checked, Obi deletes the phrases of the section after the phrase being recorded as soon as recording ends.
Do not deselect on pressing Stop twice If checked, Obi does not deselect the selection on pressing Ctrl+Space twice. If this check box is not checked, it will result in execution of the Select nothing command.
Preserve pages while recording over subsequent audio If checked, the command Record over subsequent audio does not overwrite the page phrases but retains them in the section after the newly recorded audio.
Always monitor in Recording Toolbar If checked, the Monitor Continuously check box of the Recording Toolbar is checked by default, and therefore, audio input is always monitored.
Highlight label for preview before recording If checked, Obi highlights the label of the phrase being previewed during the operation Preview before recording.
Use Play button and Space key to play section If checked, the Play command and the shortcut key Space play the audio of the section.
Enable quick cleanup (for network based deployment) If checked, Quick cleanup menu item in Tools menu is enabled. Quick cleanup skips the audio files where cleanup has already been performed.
Ensure cursor visibility when preview before recording is undone If checked, cursor will be visible when preview before recording is undone.
Disable creation of new headings and pages while recording If checked, the next heading and next page commands will move to the respective item and continue recording into it, but they will not create a new heading or page.
Prevent on the fly phrase detection on pages If checked, Obi will not apply phrase detection on the page phrases while live recording.
Save Audio Zoom If checked, Obi will save the current state of audio zoom across Obi sessions.

Project Tab:-

Open last project If checked, Obi opens the last project opened in previous session.
Select bookmark when project reopens If checked, Obi places the keyboard focus at the bookmarked item when the project is reopened. If unchecked, Obi will open the project in the default mode; Content view showing the first section.
Fix content view width If checked, Obi will not disturb the orientation of other visible phrases even if the length of a phrase exceeds the current width of Content view. If unchecked, Obi will alter the width of Content view with each elongated phrase and shift other phrases according to the altered width.
Optimize memory utilization If checked, optimizes the memory utilized by Obi.

Show waveforms in Content view If checked, Obi displays the waveforms beneath the phrase labels in the Content view. If unchecked, the waveforms are hidden and the length of the contained audio is displayed instead.
Always ignore indentation for DAISY export files If checked, the exported files (such as SMIL, OPF, etc.) will not be indented for human readability.
Highlight empty sections in TOC view If checked, Obi displays the empty sections in pink background to distinguish them from others.
Save location and size of Obi If checked, Obi saves the current size of the application window and its location on the screen, and reopens every session with the saved display parameters.
Keep the peak meter aligned to right edge of Obi If checked, the Graphical Peak Meter will be kept aligned to the right edge of Obi window. Its position will not be changed even if the user tries to move it. Please note that this option will not work if the Obi window is maximized.

Do not minimize Obi with activation of Recording Toolbar If checked, Obi window is not minimized when user opens the Recording Toolbar.
Enable Mouse wheel scrolling Lets you enable or disable mouse wheel scrolling for Content view.
Disable collapse of selected section in TOC View If checked, double-clicking the selected section in TOC view does not collapse or expand it.
Open Obi in maximize state Opens Obi application window in the maximized state.
Use Vision Australia rule for XHTML export If checked, the XHTML export is done according to the rules specified by Vision Australia.
Enlarge phrases as per increase in screen resolution If checked, Obi enlarges the phrases with the increase in screen resolution to preserve their orientation in the Content view. This results in improved efficiency as it prevents accumulation of large number of waveforms in the view.
Fix TOC View width If checked, On resizing of Obi TOC View width remains fixed.
