Skippable notes

Skippable notes are the set of contiguous phrases containing the information that may be skipped or included, as per the user’s choice, while reading the book. They can belong to any skippable element of a book such as footnote, annotation, sidebar, etc. In a physical book, a word or phrase is tagged by “*” or a numeral to notify that it points to a book element. Likewise, you can mark any phrase as anchor in Obi and associate it with the desired skippable note.

 

You can create and link skippable notes using new Obi and export them to a DAISY 3 book. Obi also imports skippable notes while importing audio-ncx as well as full-text, full-audio DAISY 3 book. However, this feature is not supported for DAISY 2.02 export.

 

Creating a skippable note and associating it with an anchor

The commands pertaining to the creation of a skippable note and associating it with an anchor can be found in the Skippable notes item under the Phrases menu of the menu bar. But you can use the context menu and keyboard shortcuts to speed up the process. The following instructions exemplify the process of creating a skippable note via one kind of skippable note - annotation:

  1. Use the arrow keys or mouse click to land on the phrase from where annotation should begin.
  2. Right click on the phrase or press the applications key on the keyboard to invoke the context menu and select Skippable note->Begin note therein. Alternatively, you can use the shortcut key combination - Ctrl+Alt+[. This will mark the beginning of the annotation.
  3. Go to the last phrase to be included in the annotation. If this is also the last phrase of the section, you can simply press the shortcut key End to reach there.
  4. Select Skippable note->End note from the context menu or press Ctrl+Alt+] to mark the end of annotation. An Assign note dialog opens.
  5. Choose annotation from the combo box therein.
  6. Click on the OK button or press Enter to classify the selected set of phrases as annotation.
  7. Go to the phrase that will serve as an anchor.
  8. Select Skippable notes->Add reference from the context menu or press Ctrl+Alt+A. The Add/Remove reference dialog opens.
  9. The first box will display information about the selected phrase and second will list the skippable notes available in your project.
  10. From the list of skippable notes, select the annotation just created using the Up and Down arrow keys.
  11. Click on the Associate button to link the phrase with the selected annotation. The selected phrase will thus become anchor and will point to the annotation. This will also activate the Dissociate button that will enable removal of the linkage.
  12. Press OK to return to the application window.

The selected phrase will now contain an extra tag “Anchor”. Please note that if you dissociate its association, the phrase will start showing “anchor= ?” to notify that there is no reference corresponding to it.

 

Editing multiple associations

While inspecting and editing the book contents, you may have to rectify multiple associations. This can be accomplished by using the Add/Remove reference dialog in “Show All” mode, as listed in the following steps:

1.      Click Phrases->Skippable note->Add reference from the menu bar. The Add/Remove reference dialog opens.

2.      Click on the Show All button, which will then start displaying Show selected. Please note that if the button is already displaying Show selected, this means that the dialog has been opened in the “Show all” mode.

Now, you can dissociate or associate any skippable note with the anchor. The left box lists the anchors and the box on the right displays the list of skippable notes in your project. An “=” sign following the anchor denotes that it is associated with some skippable note.

 

To remove the association, simply select the anchor and click on the Dissociate button. But to associate, you must select both the anchor and the required skippable note, and then click on the Associate button.

 

Navigating to or within a skippable note

You can navigate to a skippable note, referred by the selected anchor, by clicking Skippable note->Goto referred note on the Phrases menu.

 

Navigation commands to move to the first or last phrase of the skippable note (if you are within the skippable note) are:

  • Phrases->Skippable note->Move to start note or Alt+Home to go to the first phrase of the current skippable note.
  • Phrases->Skippable note->Move to end note or Alt+End to go to the last phrase of skippable note.

Please note that you can use the context menu also for accomplishing the above tasks.

 

Removing an association

An association can be removed by selecting it, and subsequently, clicking Phrases->Skippable note->Remove reference on the menu bar. You can also use the context menu for the same or press Ctrl+Alt+D.

 

Alternatively, you can dissociate the referred note from its anchor by using Add/Remove Reference dialog in the “Show all” mode as described in Editing multiple associations.

 

Clearing role of phrases contained in a skippable note

You can clear the role of all the phrases of a skippable note by a single command. Follow the steps listed below to clear the role:

  1. Go to any phrase within the boundaries of a skippable note that you wish to remove.
  2. Click Skippable note-> Clear role from note on the Phrases menu or context menu.

Obi will clear the role of all the phrases that form the current skippable note.

 

Fine navigation

Fine navigation maps Left and Right arrow keys to the backward and forward nudge operation, thereby making the editing process simpler. Follow the steps listed below to understand how fine navigation works.

  1. Start the playback of the phrase by pressing Space.
  2. Again press Space to pause.
  3. Press F2 to turn on fine navigation.
  4. Now use Left and Right arrow keys to nudge backward or forward respectively.

If you are working directly on the waveform, you can turn on fine navigation by first marking a position by clicking on the waveform and subsequently pressing F2.

 

The menu item Transport->Navigation->Fine navigation toggles with each click and informs that fine navigation is turned on by a preceding visual tick mark.  You can turn off fine navigation by pressing Shift+F2 or Esc. Obi will turn off the fine navigation automatically if you perform audio manipulation operations such as cropping or splitting the audio.

 

Please note that you can also modify the nudge duration, which determines the step size for nudging, using the audio preferences tab of Preferences dialog.

 

Customize the colors used in Obi’s GUI

In addition to customizing keyboard shortcuts to personalize Obi, you can now set the colors of various elements of Obi’s GUI by using the new tab of Preferences dialog, named Color preferences.

 

It presents a list of all the graphical elements of Obi’s application window, and other controls to choose the desired color. To change the color of a graphical element, follow the steps listed below:

  1. Open Preferences dialog by clicking Tools->Preferences on the menu or by pressing Ctrl+Alt+F.
  2. To go to the Color preferences tab, press Ctrl+Tab repeatedly.
  3. Select the graphical element for which the color has to be changed.
  4. Choose the desired color from the Assign Color combo box.
  5. If you wish to choose a different color for high contrast mode, it can be done using Assign High Contrast color combo box.
  6. Click on the OK or Apply button for the changes to take effect.

 

Extended Go to operation

The Go to operation has been extended for quick and easy navigation to any special phrase (phrases other than plain phrases) in the project. The new menu item, Collect Special Phrases, in the submenu of Go to… enables you to collect phrases of one sort and search for the desired phrase(s) from the filtered list. Follow the steps listed below to learn how to use this feature:

  1. Click Phrases->Go to…->Collect Special Phrases on the menu bar or press Ctrl+F3.
  2. From the combo box, listing special roles, select the type of phrases you wish to find; for instance, select “annotation” to find all the skippable notes of annotation type.
  3. Click the Find button.
  4. The list box underneath the combo box and Find button will display all the phrases of the selected type.
  5. Select the phrase to which you want to navigate and click OK.

 

Append section name to audio file name while exporting

A new parameter is introduced in the export operation that will instruct Obi to append section names to the name of audio files. It will help in determining which section’s audio content is stored in the audio file. This will especially be useful when the DAISY book has to be played using MP3 player, as the audio file names will be more meaningful now.

 

To add section names to audio files, check the checkbox Append section name to audio file name in the export dialog. The name of audio files will thus look like “aud001_SectionName.mp3”. Please note that the symbols like [, *, etc. should be avoided in section names since the resulting audio file name will not be valid as per DAISY specifications; these characters are not supported in file names.

 

Enhanced Time display control

The time display control of Transport bar has been enhanced to provide more relevant data. The options of the time display vary according to the state of Transport bar and are described below.

While recording:

  • Elapsed: Time elapsed since the recording was initiated.
  • Elapsed in section: Time past the beginning of the audio content of the enclosing section.
  • Elapsed in project: Duration of audio from the beginning of the audio content of the project to the current recording position.

During playback:

  • Elapsed in phrase: Length (in time) of the audio content from the beginning of the phrase to the current playback position.
  • Elapsed in section: Length (in time) of the audio content from the beginning of the enclosing section to the current playback position.
  • Elapsed in project: Length (in time) of the audio content from the beginning of the project to the current playback position.
  • Remaining in phrase: Duration of audio content from the current playback position to the end of the current phrase.
  • Remaining in selection: Duration of audio content from the current playback position to the end of the selection.

Additional information

The Assign note / Renumber pages dialog can also be used to renumber pages enclosed within the boundaries set by the Begin note and End note commands. The radio button Renumber pages in the Assign note dialog enables this functionality. Simply choose this radio button and click OK. It will open Set page number dialog, where you can provide inputs for renumbering page phrases within the selection.
