What's New?

Obi 5.1 beta comes with enhanced functionality by using artificial intelligence, incremental features and bug fixes. The synthetic speech generation has been enhanced by introduction of high-quality Azure AI voices from Microsoft, noise reduction has been enhanced by using AI models, performance improved for phrase navigation along with other improvements.

Version 5.1 beta will work on Windows 7, 8, 10 and 11. Please use version 4.8 for Windows Vista.

Highlights of Obi 5.1 Beta release

Introduced high quality Microsoft Azure AI voices for synthetic speech generation

Noise reduction improved by using AI models

Other Improvements

Introduced high quality Microsoft Azure AI voices for synthetic speech generation

The new Obi enhances the generate speech functionality. It gives user to choose from system installed voices and Azure AI voices. Using this feature now high-quality speech can be generated from text, which is comparable with human narration. You can use this feature through Generate Audio dialog which can be opened from Transport->Generate speech->Text To Audio . To know more about how to use this functionality look into Generate Audio from the Text

Noise reduction improved by using AI models

Noise Reduction can now also be done using AI models. This will improve quality of noise reduction . There are three types of noise reduction models to choose from namely Voice, Speech and General. To quickly start using these models open Noise Reduction(AI) dialog from Tools->Audio Processing->Noise Reduction(AI). To know more about this AI approach of Noise Reduction look into Noise Reduction(AI)

Other Improvements

  • Phrase navigation improved inside a section.
  • Crashes are reduced especially in higher resolution screens.
