What's New?

Obi 6.0 beta is upgraded to the .NET 9 framework to keep Obi abreast with new technologies! This release also includes incremental feature additions and performance improvements focused on reducing overall load during navigation and improving application stability.

Version 6.0 beta will work on Windows 10 and 11. Please use version 5.1 for Windows 7/ Windows 8 and Version 4.8 for Windows Vista.

Highlights of Obi 6.0 beta release

Upgradation to .NET 9 Framework

Added Option for Specialized To-Do Behaviour

CSV Import Now Supports Section Names with Commas

Improved Efficiency and stability

Upgradation to .NET 9 Framework

Obi has been upgraded to the .NET 9 framework to ensure compatibility with the latest technologies. This upgrade also brings several improvements, including reduced startup time, improved user interface responsiveness, and better memory management.

Added Option for Specialized To-Do Behaviour

A specialized To-Do behaviour has been added to allow the creation of a small To-Do phrase while recording. This helps users easily identify and check for minor recording glitches. To enable this feature, select the Create a small phrase when TODO is marked while recording checkbox in Audio Preferences.

CSV Import Now Supports Section Names with Commas

CSV import now supports section names that contain commas. To use this feature, encapsulate the section name within double quotes. For example, if the section name is: Chapter 1, Section Name, it should be written in the CSV file as: "Chapter 1, Section Name".

Improved Efficiency and stability

Performance improvements have been made to improve overall efficiency and reduce performance-related issues. These enhancements result in smoother navigation and improved overall stability of Obi.
