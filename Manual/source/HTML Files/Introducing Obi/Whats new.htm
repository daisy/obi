Whats new?

 

Obi 5.0 beta up grades Obi to .NET 6 framework! Microsoft stopped support for .NET 4 in 2022, therefore up grading Obi to .NET 6 became top priority to keep Obi up to date with the new technologies. This beta release is mainly focused on up grading to the .NET 6 framework and resolving regression bugs. Only one enhancement is added to CSV export, for exporting comments of To-Do marks to the CSV file. Rest of the features are the same as Obi 4.8.

 

New features and enhancements of Obi 4.8 are briefed below.

New sampling rates introduced for enhanced audio quality

Trim trailing silence more efficiently

Option to customize override behavior for Preview before recording command

 

New sampling rates introduced for enhanced audio quality

This release comes with the ability to choose the sample rates 48000 Hz and 96000 Hz for your project. These higher sample rates ensure better audio quality but take more disk space. You can select these sample rates from the Audio Settings dialog while creating a new project. For information on choosing sampling rate, see the topic Audio settings.

 

Trim trailing silence more efficiently

If there is silence at the end of a section, readers will experience a lag while listening to the contents of the book. You can trim this trailing silence more efficiently from the end of sections using new commands provided in the submenu of Trim silence from section end under Sections menu or the context menu. You can apply these commands on single section, multiple sections, or entire project. For more information, see the topic Trimming Silence from the End.

 

Option to customize override behaviour for Preview before recording command

You can instruct the Preview before recording command to override audio of current phrase or audio of all subsequent phrases of the section. The Audio tab of the Preferences dialog has a new checkbox Delete following phrases while doing preview before recording that lets you disable deletion of the phrases following the phrase being recorded. For information on how to use this functionality, see the topic Preview before Recording.
