New in Obi 4.7

 

Obi 4.7 comes with the capability of importing project from a CSV file, adds the ability of importing metadata from CSV, makes CSV created by exporting Todo timings user-friendly, provides an option to export Todo timings as cue markers, along with other improvements and bug fixes.

 

New features and enhancements of Obi 4.7 are briefed below.

Import metadata from a CSV file

Create a project from a CSV file

Option to use semicolon instead of comma in the CSV file being imported

Option to export Todo marks as cue markers to WAV files

Import audio for Page while generating speech for pages

Export of Todo timings to CSV made more user-friendly

Display of Todo comments when collecting Todo phrases

 

Import Metadata from a CSV file

Obi enables you to import the key-value pairs from a CSV file and populate them in the metadata of the project. The Edit menu contains a new item Import Metadata that enables this functionality. For more information, see the topic Importing Metadata.

 

Create a project from a CSV file

Now you can import entire project from a CSV file by importing audio along with the structure. Obi now accepts full path of audio files in columns 4 through 8 of the CSV. In addition, you can provide metadata in the file named metadata.csv and store it in the same location as the CSV. Metadata will also be imported during the import. For more information, see the topic Creating a project by importing a CSV file.

 

Option to export Todo marks as cue markers to WAV file

You can now choose to export Todo marks from your project to exported WAV files as cue markers for DAISY 3 and DAISY 2.02 standards. The Export dialog has a new checkbox Write TODO timing as cue mark to enable this functionality. Please note that cue marks can be written only in WAV files. For more information on the Export dialog, see the topic Exporting the Project.

 

Option to use semicolon instead of comma in the CSV file being imported

You can instruct Obi to use semicolon instead of comma in the CSV file during the operation Creating a project by importing a CSV file. Check the checkbox Use semicolon as separator instead of comma while creating a project by importing a CSV file in the Project tab of the Preferences dialog.

 

Import audio for Page while generating speech for pages

New Obi enables you to import human voice for page labels while generating speech for pages. It is especially useful when audio is being generated for the books in languages other than English. The page numbers, however, will be converted to speech using TTS. To know more, see the topic Generating audio for Pages.

 

Export of Todo timings to CSV made more user-friendly

When you choose to write Todo timings to a CSV file in the Export dialog, they are written in both decimal and user-friendly (hh:mm:ss) formats. For more information on the Export dialog, see the topic Exporting the Project.

 

Display of Todo comments when collecting Todo phrases

Collect Special Phrases dialog also displays Todo comments when collecting Todo phrases. For information on collecting special phrases, see the topic Collect Special Phrases and Navigate.
