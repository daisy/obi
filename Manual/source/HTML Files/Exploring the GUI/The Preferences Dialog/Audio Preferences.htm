Audio Preferences

The Audio tab of the Preferences dialog enables you to configure the audio devices, the audio settings for the project, options for playback and recording, etc. These preferences are either project specific, which apply to the current project only, or Obi specific, which appertain to all Obi projects.

The preferences dialog offers the following options under this tab:

Field Field Type Specific to Description
Input device name Combo box Obi Select an audio input device (sound card input port) to be used for recording.
Output device name Combo box Obi Select an audio output device (sound card output port) to be used for playback.
Sample rate Combo box Project Select a sampling rate for recording. Higher sample rate results in better audio quality but consumes more disk space.
Audio channels Combo box Project Select mono for using a single channel or stereo for two. By default, it is set to mono to save disk space.
Audio cleanup max file size Combo box Obi Select the maximum file size of a single audio file for the data folder of the project. Bigger the audio file, better the response time; especially in the client-server architecture where projects are stored on the server and Obi fetches the data through the network.
Noise level Combo box Obi Reflects the noise level in your recording environment. Select low for a recording studio, medium for silent but not a sound proof room, or high for a noisy background. Please note that Obi sets the thresholds for peak meter in accordance with this field.
Set TTS synthetic voice Combo box Obi Select the synthetic voice to be used for audio generation of page phrases and audio clues.
Speak Button Obi Plays sample audio for the synthetic voice selected in the previous field.
Operation Combo box Obi Select the operation for which the duration has to be changed. For example, select "Nudge duration" to set the amount of time to be skipped for the Nudge command.
Duration in milliseconds Combo box Obi Set the duration or time for the operation selected in previous field.
Audio clues Check box Obi If checked, enables the audio clues, disables otherwise. Audio clues provide audio indications for non-visual users.
Use Space key to stop recording Check box Obi If checked, Obi stops recording when user presses the Space key on the keyboard. If this check box is not checked, Obi acts according to the default behavior and pauses recording.
Allow overwrite while recording Check box Obi If checked, Obi will break the selected phrase, at a position determined using a set of rules, and record in a new phrase between the two fragments. If unchecked, Obi will always create a phrase after the selected phrase.
Always record directly from transport bar Check box Obi If checked, Obi will deactivate the toggling behavior of Start Monitoring/Record button of Transport Bar and always show Record Directly in its place. As a result, each click on that button will initiate recording.
Show live waveform while recording Check box Obi If checked, draws the waveform of the audio content being recorded concurrently.
Perform phrase detection while recording Check box Obi If checked, Obi applies phrase detection to the audio being recorded on the run. At the end of recording, the recorded content will be split at the detected points.
Renumber the following pages when recording ends Check box Obi If checked, Obi renumbers the pages after the recorded audio when the record command is stopped.
Use Record button to record over subsequent audio Check box Obi If checked, pressing the Record button will result in replacing the audio content subsequent to the current position by the audio being recorded.
Enforce single audio cursor Check box Obi By checking this checkbox, you instruct Obi to use only one cursor instead of two. In this case, red cursor will automatically be mapped to the location of blue.
After recording, delete subsequent phrases Check box Obi If checked, Obi deletes the phrases of the section after the phrase being recorded as soon as recording ends.
Do not deselect on pressing Stop twice Check box Obi If checked, Obi does not deselect the selection on pressing Ctrl+Space twice. If this check box is not checked, it will result in execution of the Select nothing command.
Always Monitor in Recording Toolbar Check box Obi If checked, the Monitor Continuously check box of the Recording Toolbar is checked by default, and therefore, audio input is always monitored.
Highlight label for preview before recording Check box Obi If checked, Obi highlights the label of the phrase being previewed during the operation Preview before recording.
Use Play button and Space key to play section Check box Obi If checked, the Play command and the shortcut key Space play the audio of the section.
Enable quick cleanup (for network based deployment) Check box Obi If checked, Quick cleanup menu item in Tools menu is enabled. Quick cleanup skips the audio files where cleanup has already been performed.
Preserve Audio Zoom for Content View Check box Obi If checked, Obi will save the current state of audio zoom across Obi sessions.
Show cursor time in Transport Bar in stop state Check box Obi If checked, the Time Display field of Transport Bar shows the current cursor time in the stopped state. Otherwise, this field will always show 0 in the stopped state. Please note that this feature works only for the Elapsed in phrase and the Elapsed in Section options of Time Display.
Delete following phrases while doing preview before recording Check box Obi If checked, Obi will override entire audio of the section after the cursor position. As a result, the phrases subsequent to the current phrase will be deleted. If unchecked, Obi will only override the phrase being recorded and leave the rest of the phrases of the section as is.
Additional Audio Settings Button Obi Opens additional audio settings, which are described in the table below.

Please note that the last set of check boxes are presented in the form of a list, and therefore, can be traversed using the Down and Up arrow keys.

Additional Audio Settings

The following table lists the additional audio settings. You can activate or deactivate a setting by checking or unchecking the respective check box and clicking on the OK button. You may also discard the changes by using the Cancel button.

Field Field Type Specific to Description
Retain initial silence while phrase detection Check box Obi If checked, retains the initial silence during the process of Auto-splitting using phrase detection. If unchecked, discards the initial silence.
Use Record button to preview before recording Check box Obi If checked, plays a small portion of audio prior to the marked cursor position before initiating the Record command. If unchecked, starts recording directly.
Limit max phrase duration to 50 minutes Check box Obi If checked, sets an upper limit for the length of phrase to be 50 minutes. Otherwise, allows phrases of any length to exist.
Merge first 2 phrases detected by phrase detection Check box Obi If checked, merges the first two phrases resulting from the process of auto-splitting using phrase detection.
Fast play without pitch change Check box Obi If checked, Obi preserves the pitch while increasing or decreasing the fast play rate.
Preserve pages while recording over subsequent audio Check box Obi If checked, the command Record over subsequent audio does not overwrite the page phrases but retains them in the section after the newly recorded audio.
Disable creation of new headings and pages while recording Check box Obi If checked, the next heading and next page commands will move to the respective item and continue recording into it, but they will not create a new heading or page.
Prevent on the fly phrase detection on pages Check box Obi If checked, Obi will not apply phrase detection on the page phrases while live recording.
After playback, select the last phrase played Check box Obi If checked, Obi moves the focus to the last phrase played in the playback when playback stops. Otherwise, the focus remains on the phrase from where playback was started.
Record in first empty phrase with record section command Check box Obi If checked, the Record Section command starts recording audio in the first empty phrase of the section; otherwise, it records in a new phrase at the end of the section.
Record at the starting of section with record section command if section has only empty pages Check box Obi If checked, Obi overrides the default behavior of the Record command for the sections which have only empty pages, and starts recording from the beginning of the section. This is especially done to facilitate the narrator when the structure of the book has already been created in Obi.
Merge first 2 phrases after recording with phrase detection Check box Obi If checked, Obi merges the first two phrases resulting from the process of on-the-fly phrase detection.
Play the recorded phrase when recording stops Check box Obi Plays the audio content from the phrase just recorded. If you have recorded the content in multiple phrases, only the content from the last phrase will be played.
Remove accents from filenames during DAISY 2.02 export Check box Obi When Obi appends section names to the audio file names while exporting the project in DAISY 2.02 format, some non-English language characters can cause issues in accessing the files. If this check box is checked, Obi normalizes the file names during the export function.
Start and stop recording using single key shortcuts from TOC view Check box Obi Enables users to start and stop recording by single key shortcuts, R and T respectively, from TOC view.
Create a small phrase when TODO is marked while recording Check box Obi When TODO is marked while recording a small phrase of a second is created.

Remarks:

  • The Obi specific audio settings will remain the same for all the projects; whereas, project specific audio settings will be used for a single project.
  • If you set the audio preferences when a project is opened, the global audio preferences are overridden for that project.
  • You can set the project specific audio preferences only when the project is empty (when it does not contain any phrase).
