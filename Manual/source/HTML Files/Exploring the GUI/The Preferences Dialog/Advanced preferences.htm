Advance Preferences

The Advance tab of the Preferences dialog enables you to configure profiles for Obi. There are two types of profiles: the preferences profile and the shortcuts profile. The shortcuts profile contains the list of shortcuts to be used for various operations (keyboard shortcut preferences), while the preferences profile contains configurations of all other preferences; namely, project, audio, user profile, and color.

The fields of the Advance tab and their descriptions are as follows:

Field Field Type Description
Current Profile Read only text box The preferences profile currently loaded in Obi.
Current Shortcuts Profile Read only text box The shortcuts profile currently loaded in Obi.
Select Profile Options Group of radio buttons Select Preferences if you wish to select a preferences profile, and select Keyboard shortcuts if you wish to select a shortcuts profile.
Select Profile Combo box Select the desired preferences profile from the list of available profiles. Please note that this field gets activated only if you select the "Preferences" radio button.
Select Shortcuts Profile Combo box Select the desired shortcuts profile from the list of available profiles. Please note that this field gets activated only if you select the "Keyboard shortcuts" radio button.
Profile Description Button Opens the document giving the description of the profile chosen in the above combo box.
Project / Audio / Language / Color / Select All List of check boxes Check the check boxes for the preferences that you wish to load.
Load Button Loads the selected profile in Obi. Please note that if the check box pertinent to the set of preferences is not checked, they will not be loaded; for example, if the Project check box is not checked, the project preferences will not be loaded.
Import Button Imports the existing profile in Obi. You will be presented with the Open dialog to browse for the .xml file. Select the appropriate profile file and click Open. Henceforth, the profile will show in the list of available profiles.
Save As… Button Saves the current preference settings as a profile. You will be presented with the Save As dialog to provide the path and the name for the profile file.
Remove Button Removes the profile selected in the combo box from Obi’s list of available profiles.
Profile 1 Combo box First profile to be assigned to the Switch Profile button of the Recording Toolbar.
Profile 2 Combo box Second profile to be assigned to the Switch Profile button of the Recording Toolbar.
Assign Button Assigns both the above profiles to the Switch Profile button of the Recording Toolbar, which toggles with each click.

Let’s understand the process of loading the profile with an example. The following steps show how to load the Advanced preferences profile.

  1. In the group of radio buttons, select Preferences.
  2. From the Select profile combo box, select Advanced.
  3. Check the checkboxes pertinent to the preferences you wish to load. For example, if you want to load only the project preferences, check the checkbox Project.
  4. Click on the Load button to start loading the preferences.
Once Obi loads the selected preferences profile, its name will be shown in the Current profile text box.

Now if you wish to save the current preference settings as a custom profile, do the following:

  1. Click on the Save button. The Save As dialog will be displayed.
  2. Browse for a location where you want to save your custom profile.
  3. Enter a suitable name for the profile file.
  4. Click save.
Obi will save the current preference settings in the XML file specified by you.

Next, we will look at the process of assigning the profiles to the Switch Profile button of Recording Toolbar. Let’s suppose you wish to assign the “Basic” and “Advanced” profiles to the button.

  1. In the Assign audio profiles to Recording Toolbar group box, select “Basic” from the first combo box.
  2. Now, select “Advanced” from the second combo box.
  3. Click on the Assign button.
The Basic and Advanced profiles will be assigned to the Switch Profile button of the Recording Toolbar, which will toggle with each click and show Switch from Basic to Advanced and Switch from Advanced to Basic alternatively.
