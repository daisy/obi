Project Preferences

The Project tab of the Preferences dialog lets you modify the preferences for Obi projects, such as default projects directory, auto save interval, import tolerance, etc. It also enables you to configure the look and feel of the Obi application with options such as fixed content view width, etc.

The preferences dialog offers the following options under this tab:

Field Field Type Description
Default Projects Directory Edit box Enter a location where you want to store all your Obi projects. You can also browse for a location by clicking the Browse button next to it. This will reflect as the default location in the Location field while creating a new project.
Pipelines scripts Directory Edit box Enter the path of the directory where the pipeline scripts are kept. You can also browse for a location by clicking the Browse button next to it.
Record in local drive Check box If checked, Obi makes a local copy of the audio being recorded and copies it to the server at the end of the recording. When you check the check box, an edit box will be activated with the Browse button enabling you to provide the path of the directory where audio will be stored. This feature enhances the robustness of the record operation when working on the client server architecture.
Auto save interval in minutes Check box If checked, Obi will save the changes in a backup file after the time interval specified in the numeric edit box next to it. Please note that recording will not be disturbed even if it goes beyond this time interval, and the changes will be saved after the recording is stopped.
Import tolerance Combo box Sets the time tolerance for variation of length of the audio clip from its entry in the corresponding SMIL file. This preference is used while importing a DAISY/EPUB book. If the variation is within the limit specified in this field, Obi will import the file; otherwise, it will be shown in the error log.
Open last project Check box If checked, Obi opens the last project opened in previous session.
Select bookmark when project reopens Check box If checked, Obi places the keyboard focus at the bookmarked item when the project is reopened. If unchecked, Obi will open the project in the default mode; Content view showing the first section.
Fix content view width Check box If checked, Obi will not disturb the orientation of other visible phrases even if the length of a phrase exceeds the current width of Content view. If unchecked, Obi will alter the width of Content view with each elongated phrase and shift other phrases according to the altered width.
Enable free disk space check Check box If checked, checks for available disk space and warns the users during Record, Save, and Import operations.
Save project when recording ends Check box If checked, Obi saves the modifications to the project file as soon as recording is stopped or paused.
Show waveforms in Content view Check box If checked, Obi displays the waveforms beneath the phrase labels in the Content view. If unchecked, the waveforms are hidden and the length of the contained audio is displayed instead.
Highlight empty sections in TOC view Check box If checked, Obi displays the empty sections in pink background to distinguish them from others.
Save location and size of Obi Check box If checked, Obi saves the current size of the application window and its location on the screen, and reopens every session with the saved display parameters.
Keep the peak meter aligned to right edge of Obi Check box If checked, the Graphical Peak Meter will be kept aligned to the right edge of Obi window. Its position will not be changed even if the user tries to move it. Please note that this option will not work if the Obi window is maximized.
Timeout EPUBCheck process after 3 mins Check box If checked, Obi terminates EPUB Check Validator in 3 minutes, which is ample time for validating even a large book. This ensures that the process doesn’t hang up in case of an error.
Do not minimize Obi with activation of Recording Toolbar Check box If checked, Obi window is not minimized when user opens the Recording Toolbar.
Enable mouse wheel scrolling Check box Lets you enable or disable mouse wheel scrolling for Content view.
Open Obi in maximize state Check box Opens Obi’s application window in the maximized state.
Enlarge phrases as per increase in screen resolution Check box If checked, Obi enlarges the phrases with the increase in screen resolution to preserve their orientation in the Content view. This results in improved efficiency as it prevents accumulation of large number of waveforms in the view.
Show advanced properties in properties dialog Check box If checked, Obi shows the advanced properties, such as total file count in the project, in the Properties dialog; otherwise, these features are hidden.
Warn before section manipulation Check box If checked, Obi warns the user before performing cut, delete, or rename operations on section.
Import old NCC files with Windows 1252 encoding Check box If checked, Obi imports the NCC files with Windows 1252 encoding. This ensures that old NCC files are imported correctly.
Do not show message box while moving to phrase in another section Check box When the commands like Go to page, etc. Result in bringing the phrase of another section in Content view, Obi alerts the user by displaying a message box. You can disable the display of the warning message by checking this check box.
Switch to read only structure Check box If checked, Obi makes the structure of the book read only, thus allowing only the functions like recording and audio editing. This helps in ensuring that the narrator does not alter the structure of the book by mistake while recording audio.
Warn before deleting section Check box If checked, Obi warns the user before deleting the section.
Additional Project Settings Button Opens additional project settings, which are described in the table below.

Please note that the last set of check boxes are presented in the form of a list, and therefore, can be traversed using the Up Arrow and Down Arrow keys.

Additional Project Settings

The following table lists the additional project settings. You can activate or deactivate a setting by checking or unchecking the respective check box and clicking on the OK button. You may also discard the changes by using the Cancel button.

Field Field Type Description
Optimize memory utilization Check box If checked, optimizes the memory utilized by Obi.
Disable rollback for clean up Check box If checked, Obi will not allow the users to rollback the Cleanup command.
Check for updates Check box If checked, Obi automatically checks for new updates to the application and informs the user.
Always ignore indentation for DAISY export files Check box If checked, the exported files (such as SMIL, OPF, etc.) will not be indented for human readability.
Disable collapse of selected section in TOC view Check box If checked, double-clicking the selected section in TOC view does not collapse or expand it.
Use Vision Australia rules for XHTML export Check box If checked, the XHTML export is done according to the rules specified by Vision Australia.
Freeze TOC View width Check box If checked, Obi freezes the current width of TOC view; hence disallowing automatic or manual adjustments to it.
Use semicolon as separator instead of comma while creating a project by importing a CSV file Check box If checked, Obi uses semicolon instead of comma as a column separator in the CSV file being imported during the operation Creating a project by importing a CSV file.
Apply phrase detection while creating a project by importing a CSV file Check box If checked, Obi applies phrase detection on the audio being imported while creating a project by importing a CSV file.
