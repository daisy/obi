Color and Font Preferences

The Color and Font preferences tab of the Preferences dialog enables the users to customize the colors used for elements and components of Obi’s GUI. The figure below shows how the Color and Font preferences tab looks.

The preferences dialog offers the following options under this tab:

Field Field Type Description
Colors List List view List of GUI elements, their existing color, and the color used for them in high contrast mode. To change the colors for an element, select it herein; thereafter, select the colors using the next two combo boxes.
Assign color Combo box List of available colors; choose the preferred color for the selected item.
Assign high contrast color Combo box List of available colors; choose the color that should be used for the selected item in high contrast mode.
Choose font Combo box Select the font that you want to use for the Obi’s UI.
Appearance of the select font Text box Key in some text to see how the selected font looks.
