The Preferences Dialog

Preferences are the settings and adjustments that will be used in all sessions of Obi. They are of the following types:

  • Project preferences
  • Audio preferences
  • User profile preferences
  • Keyboard shortcut preferences
  • Color and font preferences
  • Advance preferences

Mostly, the preferences are global in Obi, and thus are used for all projects. Some of the audio preferences, however, can be chosen for a single project, which will be used locally for the project in which they were selected; in all other projects, global preference settings will be used.

You can set or modify preferences by clicking Tools->preferences on the menu or by pressing Ctrl+Alt+F. A dialog box opens with six tabs (pages), one for each type of preference.

Remarks:

  • You can click on the Apply button to save the preferences modified in the current tab without exiting the Preferences dialog.
  • If you wish to restore the default preferences, simply click on the Reset button. Please note that this command will restore the default values of preferences listed in the currently visible tab only.
