The Menu Bar

Obi provides an exhaustive set of menu items. The menu is designed to align with the conventions used in popular Windows applications. The familiar environment along with the logical grouping of menu items ensures easy access to the commands of Obi.

The topics listed below unfold the operations performed by each menu. Every topic contains a table listing the items contained in the respective menu, their keyboard shortcuts and brief descriptions.

The menu items are classified as:

  • File menu
  • Edit menu
  • View menu
  • Sections menu
  • Phrases menu
  • Transport menu
  • Tools menu
  • Help menu
