Todo Comment

Provides commands for adding, editing, or removing Todo comment from selected phrase.

It has a submenu listing the following commands:

  • Add/View Comment: Enables you to add a Todo comment on a phrase or a location inside a phrase, or view or edit a Todo comment placed earlier.
  • Clear Comment: Removes the Todo comment .
