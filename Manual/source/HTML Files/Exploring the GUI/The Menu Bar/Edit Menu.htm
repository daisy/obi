Edit Menu

Edit menu allows you to perform the edit operations in Obi, like undo previous changes, redo the undone changes, cut, copy, paste and delete an item such as section, phrase, audio and text. It also provides commands to set a bookmark or navigate to it. Besides this, it facilitates searching for an item.

The Edit menu lists the following commands.

Menu Item Keyboard Shortcut Description
Undo Ctrl+Z Reverses the last executed command.
Redo Ctrl+Y Re-executes the last undone command.
Cut Ctrl+X Deletes the selected item and places it in the clipboard. Please note that you cannot cut metadata entries. However, you can cut text from the entries if validation permits you to do so.
Copy Ctrl+C Copies the selected item and places it on the clipboard. You cannot copy metadata entries, though you can copy the text from the entries.
Paste Ctrl+V Pastes the item from the clipboard in place of the Selection or after it.
Paste before Ctrl+Alt+V Pastes the section from the clipboard before the current section at the same level. Please note that this command can be executed only in the TOC view.
Paste inside Ctrl+Shift+V Pastes the section from the clipboard as the last subsection of the current section. Please note that this command works in TOC view only.
Delete Del Deletes the selected item (section, phrase, audio selection, metadata entry or text). Please note if you delete a section in TOC view, all its subsections will also be deleted, but if you delete a section in the Content view, all the subsections will be brought to the level of the deleted section.
Also you can only delete a metadata entry if its deletion does not render the project invalid.
Select nothing N.A. Deselects the selected item.
Delete unused data Ctrl+Shift+L Deletes all the sections and phrases that are marked as Unused in the project.
Import Metadata - Enables you to import key-value pairs of metadata from a CSV file and populate them in your project. For more information, see the topic Importing Metadata.
Find Ctrl+F Opens the Search bar to key-in the text and find the item containing the entered search text.
Find next F3 Finds the next item containing the search text.
Find previous Shift+F3 Finds the previous item containing the search text.
Bookmark N.A. Opens a sub menu to set the current item as bookmark or navigate to one.
