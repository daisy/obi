Apply phrase detection…

 

Splits all the phrases of the selection using phrase detection. It has a submenu having the following items:

  • Selected Item: Splits the selected phrase, or all the phrases contained in the selected section. The keyboard shortcut in Ctrl+Shift+P.
  • Multiple sections: Opens a dialog box that lets you select multiple sections whose phrases should be split using phrase detection.
  • Parameters from selected silence phrase: Picks the phrase detection parameters from the selected silence phrase and stores them in audio preferences of Obi. These parameters are used in the process of on-the-fly phrase detection.

 

The following figure illustrates how the submenu looks:

Figure: Apply phrase detection... submenu

This command uses a silence phrase to determine the positions of split. The rules of looking for a silence phrase are as follows:

  • If a phrase is selected, Obi will look for a silence phrase prior to it.
  • If a section is selected, Obi will examine the first phrase of the section. If it is not a silence phrase, it will search for a preceding silence phrase.
  • If multiple sections are selected, Obi will use a silence phrase selected in the dialog Select phrase detection sections.

 

If Obi is unable to find a silence phrase, default parameters are used.
