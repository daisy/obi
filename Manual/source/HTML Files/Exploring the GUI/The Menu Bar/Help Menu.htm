Help Menu

Help menu lets you find help or report a bug.

The Help menu contains the following commands.

Menu Item Keyboard Shortcut Description
Contents F1 Opens the user reference manual of Obi.
Online resources/Issue tracker N.A. Opens Obi Wiki using the default web browser where you can download latest version of Obi, access additional documents and Issue tracker.
About Obi N.A. Shows the copyright information and the version of the product. It also lists the names of the organizations that lent their valuable contributions towards the design, development and release of the product.
Check for updates N.A. Checks for the new releases and informs the user. To disable the automatic check, check the checkbox Check for updates in the Project tab of the Preferences dialog.
