View Menu

View menu provides commands for bringing a view in or out of focus, adjusting magnitude parameters or viewing properties, etc.

The application window of Obi is highly configurable. All the views, except the Content view, can be shown or hidden by simple commands. These commands are organized in the View menu. The check box in the beginning of the menu item indicates whether the respective view is visible or not. If the check box is checked, the view is visible; otherwise, it is hidden. Plus, there are some commands to shift the keyboard focus to TOC view, Content view and Transport bar. The View menu also provides commands to magnify or shrink the text and graphics of various views.

For a detailed information on various views, see Obi's Views and Control Bars.

The View menu contains the following commands.

Menu Item Keyboard Shortcut Description
Show TOC view Ctrl+Alt+T Shows or hides the TOC view.
Show metadata view Ctrl+Alt+M Shows or hides the Metadata view.
Show transport bar Ctrl+Alt+B Shows or hides the Transport Bar.
Show status bar Ctrl+Alt+U Shows or hides the Status Bar.
Focus on TOC view/Content view F6 Moves the keyboard focus to Content view if it is presently in TOC view, and moves the focus to the TOC view if the focus is in any other view.
Focus on transport bar Ctrl+F6 Moves the focus to the Transport Bar.
Show peak meter Ctrl+Alt+P Shows or hides the graphical Peak Meter.
Show peak meter inside Obi N.A. Shows or hides miniature view of peak meter between the TOC view and the panel splitter that separates TOC view from Content view.

Show source view Ctrl+Alt+O Shows the source file of the project. This feature should be used very cautiously as a small error may render the project invalid.
Show recording toolbar Ctrl+Alt+R Invokes the Recording toolbar.
Zoom in Ctrl+Alt++ Increases the size of all the elements on the Obi screen.
Zoom out Ctrl+Alt+- Decreases the size of all the elements on the Obi screen.
Normal size Ctrl+Alt+0 Brings all the items on the Obi screen to their normal sizes.
Audio zoom in Ctrl+Alt+Shift++ Expands the waveforms of the phrases visible in the Content view horizontally (along the x-axis). Thus, each waveform will be enlarged with respect to time without affecting its amplitude. Please note that this command will produce only a visual effect.
Audio zoom out Ctrl+Alt+Shift+- Decreases the size of the waveforms visible in the Content view horizontally (along the x-axis). Thus, each waveform will be contracted with respect to time without affecting its amplitude. Please note that this command will produce only a visual effect.
Reset audio size Ctrl+Alt+Shift+0 Returns the waveforms zoomed previously to normal size.
Section/Phrase properties Alt+Enter Displays the properties of the selected section or phrase. This menu item changes with selection (section/phrase).
Project properties N.A. Shows the properties of the project such as the title of the project, the number of sections, etc.
