Executing Commands on Multiple Phrases

Obi enables you to select a set of contiguous phrases and perform various operations on them. To perform operations on multiple phrases, do the following:

  1. Select the first phrase from which selection should begin.
  2. Click Phrases->Multi phrase/page operation->Begin mark on the menu.
  3. Go to the last phrase of the range and click Phrases->Multi phrase/page operation->End mark on the menu. A dialog box will appear enabling you to select an operation to be performed as shown below.

  4. Select the desired operation and click OK.

The Multi-phrase operations dialog presents the following options

  • Skippable notes: Mark the selected phrases as skippable notes. You can choose the desired role from the combo box in the dialog thus opened.
  • Renumber pages: Renumber the page phrases in the selected range. This option opens a dialog box asking you the details like the initial page number, page-kind, etc.
  • Time elapsed: Calculate the total duration of audio content in the selected set of phrases.
  • Change volume: Increase or decrease the volume of selected content using the audio processing functionality.
  • Normalize: Normalize audio contained in the selected range of phrases using the audio processing functionality.
  • Speech Rate: Increase/Decrease the speech rate of audio contained in the selected range of phrases using the audio processing functionality.
  • Noise Reduction: Reduce the noise in the selected range of phrases using the audio processing functionality.
  • Noise Reduction: Reduce the noise in the selected range of phrases using the audio processing AI models.
  • Cut: Remove the selected phrases and place them in the clipboard.
  • Copy: Copy the selected phrases to the clipboard.
  • Delete: Delete the selected phrases.
  • Merge: Merge the selected phrases into one.

The Cut and Copy commands mentioned here place the selected phrases in a separate clipboard and therefore require a special Paste command, Paste Multiple Phrases, which can be accessed via the submenu of Multi-phrase Operations in the context menu, or by pressing Ctrl+Alt+Z. The phrases will be pasted as per the following rules:

  • If a section is selected, the phrases will be pasted at the end of the section.
  • If a phrase is selected, the phrases will be pasted after the selected phrase.
  • If a phrase separator is selected, the phrases will be pasted at its place.
