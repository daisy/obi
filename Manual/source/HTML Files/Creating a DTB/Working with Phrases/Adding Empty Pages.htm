Adding Empty Pages

You can add empty pages in your project by following the steps listed below:

  1. Click Phrases->Add Empty Pages on the menu or press Ctrl+Shift+B. a dialog box opens as shown in the figure below.

  2. Enter the page number for the first page of the set of pages to be created in the Page number text box. Obi suggests a page number in this field, 1 more than the previous one.
  3. In the Number of Pages text box, specify the number of page phrases to be created.
  4. Select the type of pages to be created from the Page kind combo box: page-front, page-normal or, page-special.
  5. Click OK.

Obi creates new page phrases after the selected phrase or as the last set of phrases of the selected section.

During this operation, you can also perform the operations Filling Missing Pages and Renumber Following Pages by checking the check boxes Auto fill pages and Renumber following pages respectively in the Set Page Number dialog.
