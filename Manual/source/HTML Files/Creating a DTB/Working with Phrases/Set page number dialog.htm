Set Page Number Dialog

The Set Page Number dialog is used in various page operations; such as adding empty pages, assigning a page role, renumbering following pages, etc. The dialog is illustrated in the figure below.

The fields of the dialog are as follows:

Field Field type Description
Page number Text box Page number for the selected phrase of the first phrase to which the page operation will be applied. Obi suggests a page number in this field, 1 more than the previous one.
Number of pages Text box Number of page phrases to be created. Please note that this field is only relevant to the Add Empty Pages command.
Page kind Combo box Type of page: page-front, page-normal or, page-special.
Auto fill pages Check box Identifies the gaps in the page numbers and creates the missing ones automatically.
Renumber following pages Check box Renumbers the pages subsequent to the current cursor position to correct their sequence.
