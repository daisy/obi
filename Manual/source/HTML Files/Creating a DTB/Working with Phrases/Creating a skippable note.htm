Creating a Skippable Note

There are various types of skippable notes but all of them are created using the same process. The following steps exemplify the process of creating a skippable note of annotation type using context menu and keyboard shortcuts. You may also use the commands listed in the submenu of Phrases->Skippable note…, which are similar to those listed in the submenu of Skippable notes in the context menu.

  1. Use the arrow keys or mouse click to land on the phrase from where annotation should begin.
  2. Right click on the phrase or press the applications key on the keyboard to invoke the context menu and select Skippable notes->Begin note therein. Alternatively, you can use the shortcut key combination Ctrl+Alt+[. This will mark the beginning of the annotation.
  3. Go to the last phrase to be included in the annotation. If this is also the last phrase of the section, you can simply press the shortcut key End to reach there.
  4. Select Skippable notes->End note from the context menu or press Ctrl+Alt+] to mark the end of annotation. An Assign note dialog opens as shown in the figure below.

  5. Choose annotation from the combo box.
  6. Click on the OK button or press Enter to classify the selected set of phrases as annotation.

All the phrases selected in the range will be tagged as annotation and will be included in the list of skippable notes of your project. Once created, the skippable note can be associated with any phrase of the project as described in Associating Skippable Note to Anchor.

Please note that you can mark the beginning and end of a skippable note on the same phrase also.
