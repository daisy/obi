Phrase Properties

You can see or modify the properties of the selected phrase by clicking View->Phrase properties on the menu or by pressing Alt+Enter. The Phrase Properties dialog is shown in the figure below.

It has the following fields.

Field Field type Editable? Description
Parent section title Text box Read only the name of the enclosing section.
Position Text box Read only The position of the phrase in the section.
Current cursor position Text box Read only The current position of the audio cursor in the project. In other words, time elapsed since the beginning of the project.
Ancestor section (s) List box Read only Lists the name of parents of the enclosing section, starting from the top-level.
Duration Text box Read only The total duration of audio content in the phrase.
Phrase role Combo box Read / Write The role assigned to the phrase: “Heading” indicates that the phrase contains audio of the heading of the section; “page” informs that it is a page phrase; “Plain” signifies that the phrase contains no special role; “Silence” indicates that the phrase contains the audio of silence of the recording environment; "Custom" signifies that the phrase holds a user-specified role.
Custom class name Text box Read / Write Sets the class name for the custom role. Please note that this field is enabled only if you select "Custom" in the previous combo box.
Page number details Text box Read only Displays the page kind and the page number of the selected page phrase. Please note that this field is visible only if the selected phrase has a page role.
Change page number Check box Read / Write If checked, Obi opens the Set page number dialog after you press OK. Please note that this field is visible only if the selected phrase has a page role.
File name Text box Read only Displays the name of the file containing the audio content of the phrase. This advanced property is displayed only when the project preference Show advanced properties in properties dialog is enabled.
Used Check box Read / Write If checked, the phrase is a Used phrase, or else it is an Unused phrase.
Todo Check box Read / Write If checked, the phrase is marked as Todo phrase.
