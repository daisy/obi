Filling Missing Pages

You can create missing pages in your project in a single go by clicking Phrases->Multi phrase/page operations->Auto fill missing pages on the menu.

Obi will find the gaps in the page numbers and create empty pages with appropriate page numbers in those gaps. For example, if Obi finds page number 49 after page number 45, it will create three page phrases with page numbers 46, 47 and 48 immediately after page number 45.

This command is especially useful while creating the structure of the book. You can create one page phrase at the beginning of each section and give this command to automatically create the intermediate empty pages.
