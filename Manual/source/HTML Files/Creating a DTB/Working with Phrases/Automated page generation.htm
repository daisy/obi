Automated Page Generation

Sometimes the physical books (like novels) do not have page numbers in them, but page number navigation is an important feature of DAISY standards.

The process of creating page phrases for the entire book may become quite cumbersome, but Obi enables you to do it by a single command using the Auto Page Generation dialog.

Follow the steps below to generate pages in the recorded DAISY book by an automated process:

  1. Click Phrases->Auto Page Generation on the menu. The Auto Page Generation dialog opens as shown below.

  2. In the Gaps in pages edit box, enter the minimum time interval (in minutes) after which a page should be created.
  3. Alternatively, you can choose to create a page at the end of each section by checking the checkbox Create Pages at the end of the section only.
  4. Specify the section index from where Obi should start generating pages in the Starting section index combo box. It is important to specify starting section as initial pages may contain title, copyright information, etc. for which page numbers are not required.
  5. Select Pages with synthetic speech radio button if you wish to generate audio announcing the page number, or Pages with silence radio button if you do not want to listen the announcement of page phrases while playing the book.
  6. Click OK.

Obi will create first page phrase in the section specified in the Starting section index combo box. It will create subsequent pages each after the time interval mentioned in the Gaps in pages edit box. For example, if you had entered “4” in the edit box, each page will be created after the audio content of at least 4 minutes till the end of the project.
