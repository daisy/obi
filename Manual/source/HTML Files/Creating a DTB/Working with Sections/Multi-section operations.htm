Multi-section Operations

 

Obi empowers you to execute various commands on multiple sections in a single go. The dialog box multi-section Operations lets you select a range of sections and perform an operation, such as Merge, Increase Section Level, etc., on the selected range. It also provides quick glance at the structure of the project by listing its sections, their levels, and the pages contained in them.

 

Figure: Multi-section Operations dialog

 

The fields of the dialog box and their descriptions are given in the table below.

Field

Field Type

Description

List of Sections

List box

Provides the list of all the sections of the project starting from the selected section. Along with the name, each list item also gives more information about the section, such as its level, pages contained in it, and if it is without any audio content.

Play

Button

Plays the audio content of the heading phrase or the first phrase of the selected section. If more than one section is selected, the heading phrase of the last section is played.

Stop

Button

Stops the playback.

Select All

Button

Selects all the sections in the list.

Show Contents

Button

Displays the contents of the selected section in the Content View.

Merge

Button

Merges the sections selected in the list box. It is recommended to select the sections that are either at the same level as the first section or are its subsections. For example, if the first section is at level 2 then any of the subsequent sections in the range should not be at level 1.

Increment

Button

Increases the level of the sections selected in the range by one.

Decrement

Button

Decreases the level of the sections selected in the range by one.

Undo

Button

Executes the Undo operation to revert the project to the state prior to executing the last command.

Normalize

Button

Normalizes the audio contained in the selected sections.

Noise reduction

Button

Reduces noise in the audio contained in the selected sections.

Delete silence from section end

Button

Deletes trailing silence from the end of the selected sections.

Retain silence in the last phrase of section

Button

Finds trailing silence at the end of the section and saves it as the last phrase of the section.

Information Box

Read-only text box

Provides details of the selected range with some other useful information.

Close

Button

Closes the Multi-section Operations dialog.

 

The following steps exemplify how to execute an operation via this dialog box using the Merge function.

1.      Select the first section to be merged.

2.      Click Sections->Multi-section Operations on the menu. The Multi-section operation dialog opens.

3.       The list box will show all the sections starting from the selected section. Press the Shift key and keep on selecting the sections that should be merged using the Down Arrow key. The Information Box will report the selected range.

4.      Click on the Merge button to start merging the sections.

 

The result of this operation will be a single merged section bearing the name of the first section in the range, which will contain the phrases of all the selected sections in the same order as before.
