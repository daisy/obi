Section Properties

You can see or modify the properties of the selected section by clicking View->Section properties on the menu, or by pressing Alt+Enter. A dialog box opens as shown in the figure below.

It has the following fields:

Field Field type Editable? Description
Section title Text box Read / Write The name of the section.
Level / Depth Combo box Read / Write The level of the section. The top level section is at level 1; its immediate subsections at level 2 and so on.
Section time position Combo box Read only The current position of the audio cursor in the project, or in other words, time elapsed since the beginning of the project.
Ancestor section (s) List box Read only Shows the exact location of the section in the tree by listing the names of its parents starting from the top level.
Duration Text box Read only The total audio content in the section (in terms of time). It is the sum of audio content in all Used and Unused phrases.
Number of phrases Text box Read only The total number of phrases in the section. It includes both Used as well as Unused phrases.
Pages in section Text box Read only Comma-separated list of page numbers of the page phrases in the section.
Used Check box Read / Write If checked, the section is marked as used; otherwise it is marked as unused.
