Generating Audio from the Text

Obi enables you to generate audio from text using Generate Audio dialog.

  1. Select the phrase after which you want to generate audio. You can also select a section in which case audio will be generated after the last phrase of the section.
  2. Click Transport->Generate speech->Text To Audio on the menu this will open Generate Audio dialog as shown below.

  3. In Textbox of Generate Audio dialog you can write text for which you need to generate audio.
  4. TTS voice dropdown will help you to select TTS voice you want to use for your audio.
  5. Speech rate trackbar will help you to set the speech rate of the voice.
  6. System Voices will show TTS installed in TTS Voice dropdown. Azure Voices will show TTS provided by Azure AI.
  7. Preview will play some text written in Textbox. This will help you to select TTS as per your requirement.
  8. Clear will clear all the text written in Textbox.
  9. Generate speech will generate audio from the text written in the Textbox.
  10. Add Voice will open Add Azure Voice dialog as shown below. This will help you to add Azure AI voices to the TTS voice dropdown.

  11. Delete voice will help you to delete Azure voices from TTS voice dropdown
  12. Add Key will open Add Azure Key dialog as shown below. This will help you to activate Azure services. You need to activate Azure services for Azure AI Voices to work. Write your Azure key and region and press Add Key to activate. Note if Azure key is not written correctly or you do not activate Azure services then Azure AI voices will not work.
