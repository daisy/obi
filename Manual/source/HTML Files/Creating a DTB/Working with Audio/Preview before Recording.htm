Preview before Recording

You can define tape-like recording behavior for the Record command in Obi.

The Audio tab of the Preferences dialog contains a checkbox Use record button to preview before recording. When you check this checkbox, Obi will preview a small portion of audio prior to the cursor position before initiating recording. It will then start overriding the audio content after the cursor position.

You can instruct Obi to override only the current phrase or all the subsequent phrases in the section by checking or unchecking the audio preference Delete following phrases while doing preview before recording.
