Merging Projects

 

Obi enables you to merge several projects into one project. This feature empowers you to segregate the work among various workstations. For example, you can create a project for each chapter of the book, record the contents, and then merge them in to a single Obi project.

 

Please note that the projects that you select for merging should be created using Obi 3.0 beta 2 or later. Also, they should have the same audio settings (channels and sampling rate).

 

To merge multiple projects, follow the steps listed below:

1.      Open the Obi project that should appear in the beginning.

2.      Click File->Merge Project on the menu bar. Obi opens a dialog box to browse for the project file.

3.      Select the sub-project that should appear next in your project and click Open. Obi opens a dialog box to select more projects for merging as shown in the figure below.

Figure: Merge Projects dialog

 

The dialog box provides the following controls:

Control

Ascending Order

Button

Sorts the list of project files in ascending order of their names.

Descending Order

Button

Sorts the list of project files in descending order of their names.

 

4.      Click on the OK button to start merging.

 

Obi will merge the projects in the order they appear in the list of project files.

 

Example:

If you have a book with 10 chapters, and you have distributed it among 5 workstations each as follows:

01-project that contains chapters 1 and 2.

02-project that contains chapters 3 and 4.

03-project that contains chapters 5 and 6.

04-project that contains chapters 7 and 8.

05-project that contains chapters 9 and 10.

 

To merge these projects, do the following:

1.      Open the project 01-project.

2.      Click File->Merge project on the menu bar, and select the project file (with .obi extension) of 02-project in the dialog just opened.

3.      In the Merge Projects dialog box, click Add button repeatedly to add projects 03-project, 04-project, and 05-project in the list of project files.

4.      If they do not appear in the correct order, click on the Ascending Order button to sort them according to their number prefix.

5.      Click on OK button to merge the projects.

 

At the completion of the merge projects command, 01-project will contain the chapters 1 to 10 in the correct order.
