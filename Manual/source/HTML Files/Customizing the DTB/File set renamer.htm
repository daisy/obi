Fileset Renamer

 

The Fileset Renamer of Obi enables you to rename the files of the exported DTB, so that they have a sequential order (like audio files). This ensures compatibility with wider range of players. For example, if you rename the SMIL files using Fileset renamer, you can hear the book contents in proper order by playing them sequentially.

 

This tool also allows you to attach a prefix to the file names, such as name of your organization.

 

To rename the files, follow the steps listed below:

1.      Click Tools->Fileset renamer on the menu.

2.      Select the DAISY standard to which the exported DTB conforms in the dialog box just opened.

3.      Click on the OK button. A dialog box appears as shown in the figure below:

Figure: Fileset Renamer

 

4.      Enter the following fields:

Field

Field Type

Description

Input file

Edit Box

Full path of the OPF file (in case of DAISY 3) or the NCC document (in case of DAISY 2.02). You may also browse for the file by clicking on the Browse button.

Select the output directory

Edit Box

Enter the full path of the directory where you want to store the project with files renamed. You may also browse for a location by clicking on the Browse button.

Please note that the output directory should be different from the input directory.

The base name used for renaming

Edit Box

Enter a string to be attached in the beginning of each filename. This field can be empty, if you do not wish to provide any prefix.

List of file types to be excluded from renaming

Edit Box

Specify the file types on which Obi should not apply renaming process.

Use file system safe names

Check Box

If checked, Obi will use only the alphanumeric strings for the file names. This means that the file names will contain the characters from the character set A-Z and 0-9 only.

The maximum number of characters in the file name

Edit Box

Enter a value indicating the maximum number of characters that any file name in the DTB would contain. Obi restricts the length of all file names to this value.

 

5.      Click on the OK button.

 

Obi will rename the files and store them in the output directory specified in the dialog (as discussed above). Please note that the files of the file types provided in the exclusion list will not be renamed; they will be copied with the same names as earlier.
