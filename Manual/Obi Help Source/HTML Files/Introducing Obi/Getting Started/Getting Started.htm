Getting Started

Obi is an interactive authoring tool that facilitates creation of a digital talking book conforming to DAISY and EPUB 3 standards. Its main application window is designed to provide a convenient platform for book authoring and editing.

This section will help you setup Obi on your system. It will also give a brief on the basics of Obi. Once you get accustomed to the elements and the screen Layout, working will be much simpler.

It covers the following topics:
Installation
Starting Obi
Elements of Obi
Screen layout of Obi
