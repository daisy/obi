Starting Obi

You can start Obi by one of the following methods:

  • From Start Menu, select All Programs->Obi->Obi X.X, or,
  • Double-click on the icon named Obi X.X on your Desktop.

You will be presented by a User Registration dialog. Please take a minute to fill the form, so that you can actively participate in making Obi better. You can also choose to skip this step by clicking the Remind me later button.

You will then be presented by the Configurations dialog to key in the basic configurations for Obi. You may change the values, or go with the default values by simply clicking on the OK button.

Thereafter, an instance of Obi is started with the welcome dialog as shown below:

The dialog box offers the following options via buttons:
Button Description
Create a new project Opens the dialog box for creating a new project.
Create a new project from import Opens a dialog box for selecting a DAISY/EPUB book, XHTML file, or CSV file containing the structure of the book. On the basis of the file you chose, Obi will create a new project and import the structure and audio.
Open an existing project Opens a dialog box to browse for the project file (with .obi extension). Click on the Open button to open the project.
Open last project Starts Obi with the last project opened in the previous session.
Open Obi with no project Starts Obi with no project open in the work area.
View the manual Opens the user reference manual.
