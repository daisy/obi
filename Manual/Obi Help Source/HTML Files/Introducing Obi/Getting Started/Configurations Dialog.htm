Configurations Dialog

When you start Obi for the first time, the Configurations dialog is displayed that lets you configure the basic settings such as audio devices, profiles, etc.

The following figure shows how the dialog looks.

The fields of the dialog are described in the table below:

Field Default value Description
Input device name Primary sound capture driver The device driver used for recording audio.
Output device name Primary sound driver The device driver used for playing audio.
Select profile Basic The profile to be used for configuring preferences.
Select shortcuts profile Default shortcuts The keyboard shortcuts profile to be used by Obi.
Default Project Directory C:/Obi Projects The project directory to be used by Obi. This path will be shown by default when you create a new project. Please choose a safe location, where you will not face any write restrictions. in windows 10 you may face permissions related issues at desktop, program files and some other locations.

If you wish to retain the default values, simply click on the OK button.

This dialog appears only once, when you start Obi for the first time after installation. But you can modify these settings any time later by using the Preferences dialog.
