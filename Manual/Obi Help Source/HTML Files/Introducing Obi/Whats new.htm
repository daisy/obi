What’s new?

 

Obi 3.0 presents a new view that magnifies the waveform rapidly and enormously, empowering the users to perform the audio editing operations with ease. It also introduces the ability to merge multiple Obi projects into one.

 

Apart from these, there are some new and enhanced options to make Obi more advanced and efficient. New features and enhancements of this release are listed below:

Introduction of Zoomed Waveform view to exclusively work on waveforms

Ability to merge multiple Obi projects

Improved performance with the choice of displaying phrases without waveforms in Content view

Phrase separator can be dragged to adjust the boundaries of the phrase

Ability to save metadata for use across projects

Easy access to recording options via a new button on Transport bar

Choice of mapping Space key to the stop recording command

Advanced options for files exported in MP3 format

Option to remove indentation from exported files

Enhancements to Import Audio Files dialog

Renumbering pages made easy

 

Introduction of Zoomed Waveform view to exclusively work on waveforms

This release of Obi introduces an alternative of Content view called Zoomed Waveform view. The main attraction of this view is rapid and enormous magnification of the waveform, which enables the users to perform quick and accurate editing. It is augmented with concise but exhaustive toolbars providing the commands for audio editing, navigation, and zoom. To know more, see the topic Zoomed Waveform view.

 

Ability to merge multiple Obi projects

Obi presents the merge project feature that enables you to merge several Obi projects into one. This is especially useful when you have to distribute the work among various team members. To know how to use this feature, see the topic Merging Projects.

 

Improved performance with the choice of displaying phrases without waveforms in Content view

Content view can be configured to display or hide the waveforms as per the users’ choice. New preference, Show waveforms in Content view, is added to the Project preferences to select your preferred mode. To know more, see the topic Configuring Content view.

 

Phrase separator can be dragged to adjust the boundaries of the phrase

Now you can fine-tune the boundaries of a phrase by dragging the phrase separator. This action will split the phrase at the point where you dropped the phrase separator, and merge the clipped part with the neighboring phrase. Refer to the topic Adjusting the Boundaries of a Phrase  to know how.

 

Ability to save metadata for use across projects

Obi introduces the concept of default metadata. It enables you to save the common metadata entries at one place and reuse them across the projects by simply importing the default metadata. This feature eliminates the need of re-typing the metadata entries for each project, thereby reducing the turnaround time. Refer to the topic The Edit Area for more information.

 

Easy access to recording options via a new button on Transport bar

When you enable advanced recording mode, Obi activates the Recording Options button next to the Record button on the Transport bar. It provides the following recording options (described under Advanced Recording Mode):

·        Start monitoring

·        Record over subsequent phrases (of the section)

 

Choice of mapping Space key to the stop recording command

The default behavior of keyboard shortcut Space is to pause the recording. With new Obi, users have an option of mapping it to the stop recording command instead. Check the check box Use Space key to stop recording in audio preferences to activate this behavior.

 

Advanced options for files exported in MP3 format

New Obi lets you set the advanced options (like resample, replay gain, etc.) for encoding the MP3 files during DAISY export. If you choose to encode the audio files in the Export dialog, the Advanced button gets activated. Click this button to open another dialog box enabling you to set these advanced options.

 

Option to remove indentation from exported files

You can now choose to ignore indentation for the exported text files of DTB. Indented files are human readable, but consume more disk space. The files that are not indented, on the other hand, are smaller and are apt for online streaming. You can make your preference by checking or unchecking the check box Always ignore indentation for DAISY export files in the project preferences.

 

Enhancements to Import Audio Files dialog

The Import Audio Files dialog enlists the parameters to parse the audio file names for creating sections and their contents. Now these parameters are preceded by a check box, which gives the users the flexibility to use a particular parameter for parsing or not. In addition, the parsing algorithm is made more tolerant to slight variations of the format of audio file names, along with some other UI improvements for visibility of page phrases.

 

Renumbering pages made easy

Now you can renumber all the pages subsequent to the current phrase by simply clicking the menu item Renumber following pages on the Phrases menu. To know more, see the topic Renumbering Following Pages.
