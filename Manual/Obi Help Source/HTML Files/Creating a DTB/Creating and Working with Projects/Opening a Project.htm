Opening a Project

An existing project can be opened in Obi by following the steps given below:

1.      Click File->Open Project on the menu or press Ctrl+O. An Open dialog pops up as shown in the figure below:

Figure: Open dialog

 

2.      Select your project file (with .obi extension) and click Open.

 

Obi opens the project showing its contents in the application window. If the project preference select the bookmark when project reopens is enabled, Obi will set keyboard focus on the bookmarked item. Otherwise, it will show the first section in the Content view.

 

Recent Projects

Obi also maintains a list of recently opened projects. The menu item File->Recent Project shows the last five projects, opened in reverse chronological order with respect to date and time. Click on the project, and Obi will open it.

 

Note: You can clear the list of recent projects by clicking File->Recent Projects->Clear List.
