Setting Bookmark

 

Obi’s bookmark is very similar to the bookmarks used while reading the physical book. In a physical book, we place a bookmark at current page and start reading from there the next time without shuffling through pages. In the same way, you can mark any section or phrase as bookmark in your project and Obi will place the keyboard focus there when the project is reopened.

 

To place a bookmark in your project, follow the steps given below:

  1. Select the item (section or phrase) on which you wish to place the bookmark.
  2. Click Edit->Bookmark->Set as bookmark on the menu.

 

The selected item will be bookmarked for future reference. Whenever the project is opened the next time, the keyboard focus will be set on the bookmarked item.

 

Before closing the project, Obi will ask your consent to set the current selection as bookmark. If you accept, the selected item will be set as bookmark; otherwise, previous bookmark will remain unaffected.

 

However, if you do not wish to use bookmark, you can turn off this option using the Project preferences tab of the Preferences dialog. In this case, when the project is re-opened, the Content view will exhibit the contents of first section of the project.

 

Navigate to Bookmark

You can go to the bookmarked item anytime by clicking Edit->Bookmark->Go to bookmark on the menu.
