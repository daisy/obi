Creating a DTB

 

Obi is a tool for creating a DTB. The process of DTB creation starts with creating a project. Ideally there should be one project corresponding to each DTB. However, you can segregate the work by creating one project for each part or chapter of the book, and then merge the projects into one by using the Merge project feature.

 

 

This section explains the process of DTB Creation in detail. The process can be subdivided as follows:

 

Creating and Working with Projects

Working with Sections

Working with Phrases

Working with Audio

Exporting and Validating
