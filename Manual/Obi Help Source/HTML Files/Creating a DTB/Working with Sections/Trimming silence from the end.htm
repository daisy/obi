Trimming Silence from the End

Sometimes there is a trailing silence at the end of section, which is undesirable as the book reader will hear an unnecessary lag. Obi enables you to delete the trailing silence from the sections using this operation quickly and neatly. This functionality is provided in the Trim silence from the section end menu item under Sections menu. It lists the following commands:

  • Delete silence from end of section: Finds the trailing silence at the end of the selected section and removes it from the project.
  • Retain silence in last phrase of section: Finds trailing silence at the end of the selected section, removes it, and saves it as the last phrase of the section. Editors can then inspect this phrase to ensure if there is nothing important before deleting it from the project.
  • Delete silence from end of all sections: Removes trailing silence from the end of all sections of the project.
  • Retain silence in the last phrase of all sections: Finds trailing silence from all the sections of the project and saves them as last phrases of the respective sections.

The following list exemplifies how to execute these commands:

  1. Select a section.
  2. Click Sections->Trim silence from the section end->Delete silence from end of section on the menu.

These commands can also be applied using the Multi-sections Operations dialog for particular sections.
