Live Recording

 

Audio content can be placed into the project by recording directly in Obi.

 

There are two ways to do it:

  1. Record the entire content in a single phrase and then, perform auto-splitting. The structure editing how ever, has to be done manually.
  2. Create the structure of the book while recording using the navigation controls.

 

Before recording, you must make sure to check the microphone settings and sound level to ensure high-quality production.

 

Waveform creation on the fly

You can instruct Obi to create waveform of audio being recorded on the fly. To enable this feature check the checkbox Show live waveform in the Audio preferences tab of the Preferences dialog.

 

Auto Save

You may choose to save the project automatically after recording is stopped or paused using the Project tab of the Preferences dialog. Check the checkbox Save project when recording ends, which enables automatic saving to the project file.
