Recording into a Phrase

 

You can record into a blank phrase by selecting it and then doing any of the following:

  • Click Transport->Record->Start recording directly on the menu.
  • Press Ctrl+Shift+R.
  • Press a single key R, if the focus in Content View.
  • Click the Start Monitoring button twice on the Transport bar.
  • Click the Monitoring/Recording button twice on the Recording toolbar.
  • Press Ctrl+R twice.
  • Click Transport->Record->Start monitoring on the menu. Then, select Transport->Record->Start recording from the menu.

 

Obi will continue recording into the phrase till you stop the recording. Now, the phrase will display the duration of audio content just recorded into it, and draw a waveform of audio beneath it.

 

If you pause instead of stopping, recording will stop, and the point of recording will be saved. This means the phrase currently being recorded will be marked as a start-point for recording. Whenever you start recording again, Obi will create a new phrase next to the start-point and start recording into it. To remove the start-point, simply press Ctrl+Space.

 

If you select an item other than a blank phrase and give the Record command, Obi will create a new phrase for recording. Where the phrase will be created depends on what has been selected. Various scenarios are listed as follows:

  • If you select a non-blank phrase, Obi will create a new phrase next to it and start recording into it. But if the Allow overwrite option is turned on, Obi will split the phrase and record as per the rules described in the topic Allow overwrite option.
  • If you want to record in a phrase before the current one, create a blank phrase before it, by activating the strip cursor on its left. Now, start recording directly.
  • If you select a section, Obi will create a new phrase at the end and start recording into it.
  • If nothing is selected, a new untitled section and its first phrase will be created for recording. This section will be the last section of the project at the top level.

 

Auto-splitting

If the phrase just created is too large, it may be split into smaller ones. This can be done by using auto-splitting on the phrase. Obi will store the split phrases in the same section as the original one. You will then have to mark the structure and do some audio editing to ensure compatibility with the definition of phrases for your project.
