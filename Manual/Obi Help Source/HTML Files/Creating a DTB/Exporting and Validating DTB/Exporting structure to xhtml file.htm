Exporting Structure to an XHTML File

Obi enables you to export the structure of the project to an XHTML file by the steps listed below:

  1. Click Tools->Export as DAISY/EPUB on the menu. A dialog box appears.
  2. Check the checkbox XHTML Structure and click OK.
  3. Enter the full path of the directory where the XHTML file should be created. You can also browse for the path by clicking the Select button.
  4. Click OK.

Obi exports the structural elements (sections and pages) of the project to an XHTML file in the directory specified above. The rules for creating the XHTML file will vary according to the preference selected in the Project tab of the Preferences dialog.
