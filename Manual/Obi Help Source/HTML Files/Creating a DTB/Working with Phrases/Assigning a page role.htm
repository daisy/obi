Assigning a Page Role

You can assign a page role to any phrase by following the steps below:

  1. Select the desired phrase.
  2. Click Phrases->Assign Role-> Page on the menu or press Ctrl+Shift+G. Obi opens the Set Page Number dialog.
  3. Enter the page number for the selected page phrase in the Page number text box. Obi suggests a page number in this field, 1 more than the previous one.
  4. From the Page kind combo box, select the page kind: page-front, page-normal, page-special.
  5. Click OK.

The selected phrase will become a page phrase of the specified page-kind and the number entered in the Page number field will be assigned to it.

During this operation, you can also perform the operations Filling Missing Pages and Renumber Following Pages by checking the check boxes Auto fill pages and Renumber following pages respectively in the Set Page Number dialog.
