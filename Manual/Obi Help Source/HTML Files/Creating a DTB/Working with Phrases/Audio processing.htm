Audio Processing

Obi enables you to perform audio processing, such as normalization, amplification, etc., on the selected item. Please note that this feature is in the preliminary stage currently, and will be enhanced in the future by using professional plug-ins (like VST) in the backend.

The audio processing functions supported by Obi are:

    li>Amplify: Increase or decrease the volume of the audio contained in the selected item by a specified rate.
  • Normalize: Apply the constant amount of gain to the audio of the selected item so that the average and peak amplitudes are within the threshold of the audio file.
  • Speech rate: Compress or expand the audio content, to increase or decrease the rate at which it is played, by the specified times.

Follow the steps listed below to perform audio processing:

  1. Select a section or a phrase. You can also select a range of phrases as described after these steps.
  2. Click Tools->Audio Processing on the menu. The Audio Processing dialog will appear.

  3. Select the function you wish to apply on the audio of the selected item from the combo box Process.
  4. In the edit box Set Parameter, Enter the parameter to be used for the selected audio processing function.
  5. Click OK.

Obi will apply the specified audio processing function on the selected item. For example, if you selected the process as “Amplify” and entered the parameter as “2.0”, the contained audio will be amplified by 2.0 times.
