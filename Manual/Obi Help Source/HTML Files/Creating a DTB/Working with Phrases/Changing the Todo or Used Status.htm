Changing the Todo or Used Status

 

Obis phrases can also hold some special status. These phrases require special attention of the book editor, and are listed below:

       Todo

       Used/Unused

 

Changing the Todo status

Todo phrases are the ones on which work has to be done. There could be many reasons for marking a phrase as Todo like narrator made some mistake while narrating, some additional text needs to be inserted or deleted, etc. Such phrases can be marked Todo for inspection. Once the necessary work has been done, the Todo mark should be removed.

 

Todo mark can be applied on a phrase or a time position by clicking Phrases->Phrase is Todo on the menu or by pressing F9. To place a Todo mark inside a phrase, pause or click at the desired location and then apply Todo. When in stopped state, the Todo mark is applied on the phrase and not a particular position inside it.

 

You can also insert a Todo comment to describe the reason for marking the current location or phrase as Todo by clicking Phrases->Todo Comment->Add/View Comment. If the selected phrase is not a Todo phrase, Obi will first mark it Todo and then add the comment. The comment can be viewed or edited using the same menu item, and can be removed by clicking Phrases->Todo Comment->Clear Comment.

 

It is a good practice to mark the incomplete phrases as Todo so that you do not miss the minute details before publishing the book.

 

For information on navigating among Todo phrases, see Quick Reference to Todo Phrases. 

 

Changing the Used status

The audio of the phrase is included or excluded from the exported DTB depending on the Used status of the phrase.

 

You can set the status to Used or Unused by clicking Phrases->Phrase is Used on the menu or by pressing Ctrl+K. If the checkbox in front of the menu item is checked, the phrase is Used and vice versa.

 

Note: You cannot change the status of the phrases of an Unused section to Used.
