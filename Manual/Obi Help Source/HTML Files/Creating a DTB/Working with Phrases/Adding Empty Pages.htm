Adding Empty Pages

 

You can add empty pages by clicking Phrases->Add Empty Pages or by pressing Ctrl+Shift+B. a dialog box opens as shown in the figure below:

Figure: Set Page Number

 

The table given below gives the description of its fields:

Field

Field type

Description

Page number

Edit Box

Enter the page number you wish to give to the first page of the set of pages to be created. Obi suggests a page number in this field, 1 more than the previous one.

Number of pages

Edit Box

Enter the number of page phrases to be created.

Page kind

Combo Box

Type of page: page-front, page-normal or, page-special.

Renumber following pages

Check Box

If checked, Obi increments the page number of subsequent pages by 1. For example: If we set the first page number as 2, the next page will have the page number 3, then, the next one will have 4 and so on.
