Audio Preferences

 

The Audio Preferences tab allows you to set or modify Obi specific audio settings as well as project specific audio settings. The figure below shows how it looks:

Figure: Audio Preferences

 

The preferences dialog offers the following options under this tab:

 

Field

Field Type

Specific to

Description

Input Device Name

Combo Box

Obi

Select an audio input device (sound card input port) to be used for recording.

Output Device Name

Combo Box

Obi

Select an audio output device (sound card output port) to be used for playback.

Sampling Rate

Combo Box

Project

Select a sampling rate for recording. Note: Higher the sample rate, better the quality of audio, and hence, more disk space required.

Default sampling rate is set to 44100 hertz.

Audio Channels

Combo Box

Project

Select mono for using a single channel or stereo for two.

By default, it is set to mono to save disk space.

Operation

Combo Box

Obi

Select the duration whose time you wish to set. For example: Nudge duration to set the amount of time to be skipped for the Nudge command.

Duration in milliseconds

Slider Bar

Obi

Set the duration or time for the operation selected in previous field.

Noise Level

Combo Box

Obi

Reflects the noise level in your recording environment. Select low for a recording studio, medium for silent but not a sound proof room, or high for a noisy background.

Note: Obi sets the thresholds for peak meter in accordance with this field.

Enable advanced recording mode

Button

Obi 

Checks all the advanced options for recording from the list of checkboxes below. This button toggles with each click; it shows Enable advanced options and Disable advanced options alternatively.

Audio Clues

Check Box

Obi

If checked, enables the audio clues, disables otherwise. Audio clues provide audio indications for non-visual users.

Retain initial silence while phrase detection

Check Box

Obi

If checked, retains the initial silence during the process of Auto-splitting using phrase detection. If unchecked, discards the initial silence.

Preview before recording

Check Box

Obi

If checked, activates the Preview upto command before initiating recording. If unchecked, starts recording directly.

Start recording from cursor erasing the following audio

Check Box

Obi

If checked, deletes the audio content after the cursor position and starts recording in a new phrase next to it. If unchecked, records as per pre-defined rules.

Allow overwrite while recording

Check Box

Obi

If checked, Obi will break the selected phrase, at a position determined using a set of rules, and record in a new phrase between the two fragments. If unchecked, Obi will always create a phrase after the selected phrase.

Always record directly from transport bar

Check Box

Obi

If checked, Obi will deactivate the toggling behavior of Start Monitoring/Record button of Transport Bar and always show Record Directly in its place. As a result, each click on that button will initiate recording.

Limit max phrase duration to 50 minutes

Check Box

Obi

If checked, sets an upper limit for the length of phrase to be 50 minutes. Otherwise, allows phrases of any length to exist.

 

Please note that the last set of checkboxes are presented in the form of a list and therefore can be traversed using the Down and Up Arrow keys.

 

Remarks:

  • The Obi specific audio settings will remain the same for all the projects; whereas, project specific audio settings will be used for a single project.
  • If you set the audio preferences when a project is opened, the global audio preferences are overridden for that project.
  • You can set the project specific audio preferences only when the project is empty (when it does not contain any phrase).
