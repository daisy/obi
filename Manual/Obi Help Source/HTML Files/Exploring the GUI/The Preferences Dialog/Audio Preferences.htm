Audio Preferences

 

The Audio Preferences tab allows you to set or modify Obi specific audio settings as well as project specific audio settings. The figure below shows how it looks:

Figure: Audio Preferences

 

The preferences dialog offers the following options under this tab:

 

Field

Field Type

Specific to

Description

Input Device Name

Combo Box

Obi

Select an audio input device (sound card input port) to be used for recording.

Output Device Name

Combo Box

Obi

Select an audio output device (sound card output port) to be used for playback.

Sampling Rate

Combo Box

Project

Select a sampling rate for recording. Note: Higher the sample rate, better the quality of audio, and hence, more disk space required.

Default sampling rate is set to 44100 hertz.

Audio Channels

Combo Box

Project

Select mono for using a single channel or stereo for two.

By default, it is set to mono to save disk space.

Set TTS synthetic voice

Combo Box

Obi

Select the synthetic voice to be used for audio generation of page phrases and audio clues.

Speak

Button

Obi

Plays sample audio for the synthetic voice selected above.

Operation

Combo Box

Obi

Select the duration whose time you wish to set. For example: Nudge duration to set the amount of time to be skipped for the Nudge command.

Duration in milliseconds

Numeric Edit Box

Obi

Set the duration or time for the operation selected in previous field.

Noise Level

Combo Box

Obi

Reflects the noise level in your recording environment. Select low for a recording studio, medium for silent but not a sound proof room, or high for a noisy background.

Note: Obi sets the thresholds for peak meter in accordance with this field.

Profile

Combo box

Obi 

Enables you to select a preferred profile. Each profile will check a particular set of checkboxes as described in a help file, which can be viewed by clicking on the adjacent question mark (?).

Audio Clues

Check Box

Obi

If checked, enables the audio clues, disables otherwise. Audio clues provide audio indications for non-visual users.

Retain initial silence while phrase detection

Check Box

Obi

If checked, retains the initial silence during the process of Auto-splitting using phrase detection. If unchecked, discards the initial silence.

Use Record button to preview before recording

Check Box

Obi

If checked, plays a small portion of audio prior to the marked cursor position before initiating the Record command. If unchecked, starts recording directly.

Use Space key to stop recording

Check Box

Obi

If checked, Obi stops recording when user presses the Space key on the keyboard. If this check box is not checked, Obi acts according to the default behavior and pauses recording.

Allow overwrite while recording

Check Box

Obi

If checked, Obi will break the selected phrase, at a position determined using a set of rules, and record in a new phrase between the two fragments. If unchecked, Obi will always create a phrase after the selected phrase.

Always record directly from transport bar

Check Box

Obi

If checked, Obi will deactivate the toggling behavior of Start Monitoring/Record button of Transport Bar and always show Record Directly in its place. As a result, each click on that button will initiate recording.

Limit max phrase duration to 50 minutes

Check Box

Obi

If checked, sets an upper limit for the length of phrase to be 50 minutes. Otherwise, allows phrases of any length to exist.

Show live waveform while recording

Check Box

Obi

If checked, draws the waveform of the audio content being recorded live.

Perform phrase detection while recording

Check Box

Obi

If checked, Obi applies phrase detection to the audio being recorded on the run. After the record command is stopped, the recorded content will be split at the detected positions.

Renumber the following pages when recording ends

Check Box

Obi

If checked, Obi renumbers the pages after the recorded audio when the record command is stopped.

Merge first 2 phrases detected by phrase detection

Check Box

Obi

If checked, merges the first two phrases resulting from the process of auto-splitting using phrase detection.

Fast play without pitch change

Check Box

Obi

If checked, Obi preserves the pitch while increasing or decreasing the fast play rate.

Use Record button to record over subsequent audio

Check box

Obi

If checked, pressing the Record button will result in replacing the audio content subsequent to the current position by the audio being recorded.

Enforce single audio cursor

Check box

Obi

By checking this checkbox, you instruct Obi to use only one cursor instead of two. In this case, red cursor will automatically be mapped to the location of blue.

After recording, delete subsequent phrases

Check box

Obi

If checked, Obi deletes the phrases of the section after the phrase being recorded as soon as recording ends.

Do not deselect on pressing Stop twice

Check box

Obi

If checked, Obi does not deselect the selection on pressing Ctrl+Space twice. If this check box is not checked, it will result in execution of the Select nothing command.

 

Please note that the last set of checkboxes are presented in the form of a list and therefore can be traversed using the Down and Up Arrow keys.

 

Remarks:

  • The Obi specific audio settings will remain the same for all the projects; whereas, project specific audio settings will be used for a single project.
  • If you set the audio preferences when a project is opened, the global audio preferences are overridden for that project.
  • You can set the project specific audio preferences only when the project is empty (when it does not contain any phrase).
