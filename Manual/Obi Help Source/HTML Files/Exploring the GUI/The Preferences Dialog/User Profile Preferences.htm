User Profile Preferences

The User Profile tab of the Preferences dialog shows the user information such as User’s name, language, etc. Obi extracts this information from the operating system at the time of setup. You can, however, change these settings any time using this tab.

The preferences dialog offers the following options under this tab:

Field Field Type Description
Full Name Edit box Full Name of the user.
Organization Edit box Name of the organization.
Language Combo box Language currently used by Obi. You may set a different language for working in Obi by selecting it from the list. Please note that this change will have effect only when you restart the application.

Remarks:
In case of a language switch, Obi reconfigures the GUI only if the language opted in the dialog is supported by it. But in either of the cases, it updates the metadata entry dc:Language.
