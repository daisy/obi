Project Preferences

 

The Project Preferences tab allows you to modify the settings pertaining to a project as shown in the figure below:

Figure: Project Preferences

The preferences dialog offers the following options under this tab:

Field

Field Type

Description

Default Projects Directory

Edit Box

Enter a location where you want to store all your Obi projects. You can also browse for a location by clicking the Browse button next to it.

This will reflect as the default location in the Location field while creating a new project.

Pipelines scripts Directory

Edit Box

Enter the location of pipeline scripts. You can also browse for a location by clicking the Browse button next to it.

Obi uses the scripts from this location while performing various operations.

Auto save interval in minutes

Check Box

If checked, Obi will save the changes in a backup file after the time interval (in minutes) specified in the numeric edit box next to it.

Please note that recording will not be disturbed even if it goes beyond this time interval, and the changes will be saved after the recording is stopped.

Import tolerance

Combo Box

Sets the time tolerance for imported audio during the process of creating a project by importing a DAISY book. If the variation of duration of the audio content of a phrase from its entry in SMIL file is within the limit entered in this field, the phrase will be imported. Otherwise, it will be shown in the error log.

Open last project

Check Box

If checked, the new session of Obi opens the last project opened in the previous session.

Select bookmark when project reopens

Check box

If checked, places the keyboard focus at the bookmarked item (phrase or section). If unchecked, Obi will open in the default mode; Content view showing the first section.

Fix content view width

Check box

If checked, Obi will not disturb the orientation of other visible phrases even if the length of a phrase exceeds the current width of Content view. If unchecked, Obi will alter the width of Content view with each elongated phrase and shift other phrases according to the altered width.

Optimize memory utilization

Check box

If checked, optimizes memory utilized by Obi.

Delete unused files after clean up

Check box

If checked, Obi will delete the unused files automatically after the project cleanup process. If unchecked, presents you with an option whereby you can choose to view the unused files and delete them manually after inspecting them.

Enable free disk space check

Check box

If checked, checks for available disk space and warns the users during Record, Save, and Import operations.

Save project when recording ends

Check box

If checked, Obi saves the modifications to the project file as soon as recording is stopped or paused.

Check for Updates

Check box

If checked, Obi automatically checks for new updates to the application and informs the user.

Show waveforms in Content view

Check box

If checked, Obi displays the waveforms beneath the phrase labels in the Content view. If unchecked, the waveforms are hidden and the length of the contained audio is displayed instead.

Always ignore indentation for DAISY export files

Check box

If checked, the exported files (such as SMIL, OPF, etc.) will not be indented for human readability.

 

Please note that the checkboxes at the bottom of the dialog box are presented in the form of a list. You can traverse through them using the Up Arrow and Down Arrow keys.
