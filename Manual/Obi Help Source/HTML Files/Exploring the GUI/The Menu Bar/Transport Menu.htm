Transport Menu

Transport menu contains basic audio commands like play, pause, rewind, record, etc. It also contains commands for navigating among phrases and sections.

The Transport menu contains the following commands.

Menu Item Keyboard Shortcut Description
Play Space Lists the commands for playing audio of the selection.
Pause/Resume Space Pauses or resumes the playback. This menu item toggles with each click and shows Pause and Resume alternatively.
Stop Ctrl+Space Stops the playback.
Fast play N.A. Lists the commands for changing the rate at which audio is being played.
Preview N.A. Provides commands for playing a small portion of audio before or after the audio cursor.
Record N.A. Lists the commands for recording and audio monitoring.
Generate speech N.A. Enables generation of audio for page phrases using synthetic voice.
Navigation N.A. Provides commands for navigating in the project.
Rewind Ctrl +Comma Starts playing the audio of the phrase backwards while skipping small portions.
Fast forward Ctrl +Period Plays the audio of the phrase while skipping small portions.
