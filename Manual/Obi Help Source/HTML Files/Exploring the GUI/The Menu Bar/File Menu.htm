File menu

File menu enables you to perform basic file operations like creating a new Obi project, opening an existing one, saving, and closing it.

The File menu lists the following commands.

Menu Item Keyboard Shortcut Description
New project Ctrl+N Creates a new Obi project.
New project from import Ctrl+Shift+N Creates a new project and imports the data and/or structure (title, sections, pages, etc.) from the DAISY/EPUB book or the file containing the structure of the book. For more information, see the topic Creating and Working with Projects.
Open project Ctrl+O Opens an existing Obi project.
Save project Ctrl+S Saves the modifications made to the project.
Save project as… Ctrl+Shift+S Saves a copy of the current project.
Merge Project N.A. Merges one or more Obi projects with the current project.
Restore from backup N.A. Reopens the project using the backup file. You can then save the project to instate the backup file in place of the main project file.
Close project Ctrl+W Closes the project currently opened in Obi.
Recent project N.A. Enables you to open a recently opened project. It has a submenu that shows a list of the last five projects opened in reverse chronological order of date and time.
Exit Alt+F4 Closes the Obi application.
