Go to…

 

Simplifies navigation among phrases, so that you do not have to traverse the whole file/section to find one.

 

A submenu opens showing the following items:

  • Next Todo phrase: Selects the next Todo phrase from the current position. It goes to the first phrase, if nothing was selected. This searching wraps. This means that if the end of the project is reached, the search will start from the beginning. The keyboard shortcut is Ctrl+F9.
  • Previous Todo phrase: Selects the previous Todo phrase from the current position. It goes to the first phrase, if nothing was selected. This searching wraps. This means that if the beginning of the project is reached, the search will start from the end. The keyboard shortcut is Ctrl+Shift+F9.
  • Page/Phrase/Time: Goes to the specific page number in the project, phrase index in the selected section, or a location in audio content of a phrase/section/project. The shortcut key is Ctrl+G.
  • Collect special phrases: Finds all the phrases of one type and presents them in the form of a list. Users can then scroll through them sequentially and navigate to the desired phrase. The keyboard shortcut is Ctrl+F3.
