Audio selection

Provides commands to select a portion of audio from a phrase. A submenu opens containing the following items:

  • Begin audio selection: This marks the beginning of audio selection at the position of audio cursor. The keyboard shortcut is Ctrl+[.
  • End  audio selection: This marks the end of an audio selection at the position of the audio cursor. The keyboard shortcut is Ctrl+].
