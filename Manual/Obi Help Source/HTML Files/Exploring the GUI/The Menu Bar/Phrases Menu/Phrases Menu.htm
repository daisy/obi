Phrases Menu

 

Helps you in managing the phrases of the project. Each phrase contains some audio content. You can split or merge the phrases or mark them Used or Unused.

 

Phrases may also portray some special roles. These roles can be edited or cleared, whenever you wish to do so.

 

Obi provides some simple commands to navigate among phrases on which work has to be done.
