Sections Menu

Sections menu provides commands to add, insert, rename, or delete a section or a subsection. Additionally, you can use this menu to split or merge a section, change section levels, and mark a section as used or unused.

The Sections menu contains the following commands.

Menu Item Keyboard Shortcut Description
Add section Ctrl+H Adds a section.
Add subsection Ctrl+Shift+H Adds a new subsection as the last child of the selected section.
Insert section Ctrl+Alt+H Inserts a new section before the selected section at the same level.
Rename section F2 Allows you to change the name of a section.
Import and replace TOC N.A. Imports the structure of the book from an XHTML file and replaces the existing TOC of the project (both the section names and their levels). For more information, see the topic Importing Structure from XHTML
Decrease section level Ctrl+Shift+Left Arrow Decreases the level of the section by 1. Please note that the level of the section at the top level or the section with subsections cannot be decreased.
Increase section level Ctrl+Shift+Right Arrow Increases the level of the section by 1. Please note that the level of the first section at the top level cannot be increased.
Split section Ctrl+Shift+Q Splits the section at the selected phrase.
Merge with next Ctrl+Shift+M Merges the subsequent section with the selected section.
Multi-section Operations N.A. Enables you to perform an operation on a selected range of sections in a single go. It opens the Multi-section Operations dialog that gives access to the operations like Merge Section, Increase Section Level, and Decrease Section Level.
Trim silence from section end N.A. Has a submenu listing the commands for removing trailing silence from the end of sections.
Section is used Ctrl+K Marks the section as Used or Unused.
