Multi phrase/page operations

Enables you to select a range of phrases, as well as execute commands on multiple phrases or pages. It has a submenu listing the following commands:

  • Begin mark: Mark the selected phrase as the first phrase of the range.
  • End mark: Mark the selected phrase as the end of the range.
  • Renumber following pages: Renumbers the page phrases subsequent to the current phrase. You can specify the page number for the first phrase; the page numbers will be incremented for each following phrase thereafter.
  • Auto fill missing pages: Identify the gaps in the page phrases in the project, and create empty page phrases to fill in the gaps. For example, if page 6 follows page 3, Obi will create page4 and page 5 immediately after page 3.
  • Paste Multiple Phrases: Paste the set of contiguous phrases, cut or copied using the Multi-phrase Operations dialog, after the selected phrase or at the end of the selected section.
