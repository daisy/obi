Play

 

Starts playing the audio for the phrase. If you have selected some part of the phrase, this command starts playing from the start of the selection. It has a submenu containing the following items: 

  • Play All: Plays the audio of the project as it will be played in the exported DTB. The audio content of all phrases in the project except ones marked as Unused will be played. The keyboard shortcut is Ctrl+Shift+Space. Playing will start from the first Used phrase of the project or from the selection, if there is one.
  • Play Selection: Plays the current selection. The keyboard shortcut is Space. The selection in this context can be section(s), phrase(s) or an audio selection. You can also execute this command by pressing Play on the Transport Bar.
