Navigation

 

Provides the commands for navigation.

 

It has a submenu containing the following:

·        Previous Section: Moves to the previous section. The keyboard shortcut is Alt+Up Arrow.

·        Previous page: Moves to the previous page. The keyboard shortcut is Alt+Page Up.

·        Previous Phrase: Moves to the previous phrase in the section. The keyboard shortcut is Alt+Left Arrow.

·        Next Phrase: Moves to the next phrase in the section. The keyboard shortcut is Alt+Right Arrow.

·        Next page: Moves to the next page. The keyboard shortcut is Alt+Page Down.

·        Next Section: Moves to the next section. The keyboard shortcut is Alt+Down Arrow.

·        Play on Navigate: Plays the audio of the phrase, when a phrase receives the focus. If you move to a section, the audio of the heading phrase will be played. If the checkbox at the beginning of the menu item is checked, Obi will start playing the audio, otherwise it will just navigate.

·        Fine Navigation: Maps the arrow keys to the nudge operation, thereby simplifying the process of audio editing. The keyboard shortcut is F2.
