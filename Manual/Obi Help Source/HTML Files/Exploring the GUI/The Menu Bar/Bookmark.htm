Bookmark

 

Sets or navigates to bookmark.

 

This menu item has a submenu as follows:

  • Set as bookmark: Sets the current selection (section or phrase) as bookmark.
  • Go to bookmark: Shifts the keyboard focus to the bookmarked item.
