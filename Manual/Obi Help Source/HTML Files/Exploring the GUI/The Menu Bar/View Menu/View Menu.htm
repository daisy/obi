View Menu

Provides commands for bringing a view in or out of focus, adjusting magnitude parameters or viewing properties.

 

The application window of Obi is highly configurable. All the views, except the Content view, can be shown or hidden by simple commands. These commands are organized in View menu. The checkbox in the beginning of the menu item indicates whether the corresponding view is visible or not. If the checkbox is checked, the respective view is visible; otherwise, it is hidden. Plus, there are some commands to shift the keyboard focus to TOC/Content view and Transport bar.

 

Additionally, View menu provides commands to magnify or shrink the text and graphics of various views.

 

For more information on various views, see Obi's Views and Control Bars.
