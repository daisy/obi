Save project

 

Saves the current project and its data. The keyboard shortcut is Ctrl+S. The project will be saved in the location where it is currently placed. If there are any unsaved changes, an asterisk (*) is shown in the title bar following the title of the project.

Note: If there are no unsaved changes, this menu item will be disabled.
