Recent project

 

Allows you to open a recently opened project. It has a submenu that shows a list of the last five projects opened in reverse chronological order of date and time. Click on the path of a project, and Obi will open it.

 

The figure below illustrates how the submenu of Recent Projects looks: 

Figure: Recent Projects sub menu

Clear list

 

The submenu also shows an option to clear the list of recent projects. Once you click on it, the list will be cleared. This operation cannot be undone.
