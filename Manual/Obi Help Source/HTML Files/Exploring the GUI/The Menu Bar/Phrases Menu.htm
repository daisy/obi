Phrases Menu

Phrases menu provides the commands to edit and navigate to phrases. You can split/merge them, apply phrase detection, add empty phrases/pages, mark them used or unused, collect special phrases, and perform a number of other operations.

Phrases may also portray some special roles. These roles can be edited or cleared, whenever you wish to do so.

The Phrases menu contains the following commands.

Menu Item Keyboard Shortcut Description
Add blank phrase Ctrl+B Adds a blank phrase after the selected phrase, as the last phrase of the selected section, or at the position of the strip cursor.
Add empty pages Ctrl+Shift+B Adds blank phrases with the page role.
Import audio files Ctrl+I Imports audio file(s) into the project.
Split phrase Ctrl+Q Splits the selected phrase at the point of the audio cursor.
Merge N.A. Has a submenu listing the commands for merging any number of contiguous phrases.
Crop audio Ctrl+Alt +Q Retains only the selected audio in a phrase and removes the rest.
Phrase is Todo F9 Alters the Todo status of current phrase.
Phrase is used Ctrl+K Marks the phrase as Used or Unused.
Multi phrase/page operations N.A. Lists the commands for selecting a range of phrases, and performing operations on multiple phrases or pages.
Skippable note… N.A. Has a submenu listing the commands for working with skippable notes.
Todo Comment N.A. Has a submenu listing the commands to add, edit, or remove the Todo comment.
Auto Page Generation N.A. Generates page phrases in the recorded book in specified time gap.
Assign role N.A. Lists the roles that can be assigned to the selected phrase.
Clear role Ctrl+Shift+C Clears the role of the selected phrase.
Apply phrase detection... N.A. Splits the audio content of the selection using the phrase detection algorithm.
Audio selection N.A. Provides commands to select a portion of audio from a phrase.
Go to... N.A. Provides the commands to navigate among or within a phrase.
Check for corrupted audio N.A. Checks for the phrases with corrupted audio, and asks from the user to replace them with empty Todo phrases. Replacing them with Todo phrases will ensure that the book editor inspects them before publishing the book.
Delete following phrases N.A. Deletes all the phrases subsequent to the selected phrase in the section.
