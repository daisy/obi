Tools Menu

Tools menu provides an assortment of tools required for customization, export, validation, encoding, project clean-up etc.

Tools menu contains the following commands.

Menu Item Keyboard Shortcut Description
Preferences Ctrl+Alt+F Allows you to view or modify Obi's preferences.
Update Language Pack N.A. Enables you to update language pack in Obi. You must first download the required language pack (in ZIP format) from https://daisy.org/activities/software/obi/download-and-installation/.
Thereafter, click on this menu item and browse for the location of the downloaded zip file, and Obi will integrate the language pack into it.
Audio Processing N.A. Provides new and enhanced operations for audio processing.
Export audio Ctrl+Alt+E Exports audio contained in the selected item (section or phrase) to an external audio file.
Clean unreferenced audio Ctrl+L Deletes unreferenced audio from the disk.
Export as DAISY/EPUB Ctrl+E Exports the project as an accessible book following the DAISY 3/DAISY 2.02/EPUB 3 specifications, and structure of the book to an XHTML file.
EPUB 3 Validator N.A. Validates exported book for compliance with EPUB 3 specifications.
DAISY 2.02 Light Validator N.A. Validates exported DTB to assure compliance with DAISY 2.02 standard.
DAISY 3 Validator N.A. Validates exported DTB to assure compliance with DAISY 3 standard.
Convert audio to MP3 N.A. Encodes audio of the exported DTB in MP3 format.
Fileset renamer N.A. Renames the files of the DTB, according to the specified convention.
