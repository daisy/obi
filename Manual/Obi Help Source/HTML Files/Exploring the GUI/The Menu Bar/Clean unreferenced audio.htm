Clean Unreferenced audio

Provides commands to clean audio that is not referenced in the project. It has a submenu listing the following commands:

  • Complete cleanup: Removes all the unreferenced data from the project. The keyboard shortcut is Ctrl+L.
  • Quick cleanup: Skips the audio data that has already been cleaned and removes the unreferenced audio from the rest. The keyboard shortcut is Ctrl+Alt+L. It is particularly useful in the network based deployment of Obi. You can enable or disable this menu item using the Audio tab of the preferences dialog.
  • Rollback: Reverts the last cleanup operation.
