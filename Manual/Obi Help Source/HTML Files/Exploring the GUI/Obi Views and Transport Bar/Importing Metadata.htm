Importing Metadata

Obi enables you to import metadata from a CSV file. The CSV file should contain key-value pairs of the metadata entries. To know about Obi's metadata entries, see the topic List of Metadata Entries

The table below exemplifies how to create the metadata CSV file:

Key Value
dc:identifier ff8157777c46
dc:lang en-us
dc:publisher Notion Press Ltd.
dc:title Book of Shorts

To import metadata from CSV, do the following:

  1. Click Edit->Import Metadata on the menu. Obi will open the file explorer browser.
  2. Browse for the CSV file containing metadata and click on Open button.

Obi will import the key-value pairs and populate the metadata entries of your project. Any existing entries will be overwritten.
