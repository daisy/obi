Audio and Navigation Controls

 

Transport bar offers many button controls to perform actions like playback, recording and navigation.

 

The table below provides information on these audio and navigation controls. The first column of the table suggests, whether the control is used for audio playback / recording (A), both audio and navigation (B) or for marking (M).

 

Control

Control type

Keyboard Shortcut

Description

A

Play

Button

Space

Starts or resumes playback of the selection. If nothing is selected, the whole book is played.

It acts as Play button, if Transport Bar is in stopped state; Resume button, if Transport Bar is in paused state.

A

Pause

Button

Space

Pauses or resumes the playback or recording.

A

Stop

Button

Ctrl+Space

Stops the playback or recording.

If Transport Bar is already in the stopped state, deselects the selection.

A

Start Monitoring  / Record

Button

Ctrl+R

Starts monitoring, when pressed for the first time.

Starts recording, when you press again. Please note that if the advanced recording mode is enabled, each click on the button will result in initiation of recording.

 

A

Rewind

Button

Ctrl+Comma

Starts playing backwards skipping small parts of audio.

A

Fast Forward

Button

Ctrl+period

Starts playing forward skipping small parts of audio.

A

Fast Play Rate

Combo Box

F for faster / S for slower

Specifies how much faster the audio should be played.

For example, if it should be played 1.5 times faster, select 1.5 from the combo box.

If you use keyboard shortcut, pressing f will select the next value in the combo box, and pressing s will select the value above the current one.

B

Next Phrase

Button

Alt+Right Arrow

Moves to the next phrase of the current section.

If Transport Bar is in playing state, jumps to the next phrase in the play list.

If Transport Bar is in recording state, Obi creates a new phrase after the current phrase and starts recording into it. 

B

Previous Phrase

Button

Alt+Left Arrow

Moves to the previous phrase of the current section.

If Transport Bar is in playing state, moves to the previous phrase in the play list. But, if the audio cursor goes beyond a certain threshold time, moves to the beginning of the current phrase.

B

Next Section

Button

Alt+Down Arrow

Moves to the first phrase of the next section.

If Transport Bar is in recording state, creates a new untitled section after the current section and starts recording the first phrase.

B

Previous Section

Button

Alt+Up Arrow

Moves to the first phrase of the previous section.

If Transport Bar is in playing state, jumps to the previous section in the play-list.

B

Next Page

Button

Alt+Page Down

Moves to the next page phrase.

If Transport Bar is in recording state, creates a phrase after the current one, with the page number incremented by 1 than the closest preceding page phrase, and starts recording into it.

B

Previous Page

Button

Alt+Page Up

Moves to the previous page phrase.

If Transport Bar is in playing state, jumps to the previous page phrase in the play-list.

M

Mark Phrase

Button

F9

Marks the current phrase as to do.

If Transport Bar is in recording state, marks the current phrase as to do and creates a new phrase after the current one, and starts recording into it.
