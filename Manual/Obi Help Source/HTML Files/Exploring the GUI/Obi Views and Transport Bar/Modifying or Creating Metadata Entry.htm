Modifying or Creating a Metadata Entry

 

This topic describes how to:

  • Modify a metadata entry
  • Create a new metadata entry

 

Modifying metadata entries

Any metadata entry can be modified except the required ones. Follow the steps given below to modify a metadata entry:

  1. Select an entry from the list using the mouse or keyboard.
  2. Press Tab to go to the edit area.
  3. Enter the contents.
  4. Click on the Update button to submit the changes.

 

If the checkbox at the beginning of the entry was unchecked earlier, Obi will check it; the entry will be marked for inclusion.

 

Creating a new metadata entry

You can also create a new entry by following the steps given below:

  1. Select the entry named Custom from the list of metadata entries.
  2. Press Tab to go to the edit area.
  3. Enter the name and content.
  4. Click the Update button to submit the changes.

 

A new entry appears in the list before the custom entry.
