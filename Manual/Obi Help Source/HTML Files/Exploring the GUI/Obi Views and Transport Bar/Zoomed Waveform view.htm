Zoomed Waveform view

 

Zoomed Waveform view is an alternative of Content view that lets you work on the waveforms conveniently. It displays the magnified waveform of a single phrase or an audio selection at a time. As it provides enormous and rapid zoom of waveforms of any size, it assists in quick and accurate editing.

 

At the bottom of the Zoomed Waveform view, you will be able to find information bar and two toolbars that provide easy access to the navigation, zoom, and edit commands. Apart from these, the Content view keyboard shortcuts for audio selection and editing can also be used. A word of caution here: some single key shortcuts may be trapped when the focus is on the toolbars; it is recommended to first move to the Information bar, and then use the Content view keyboard shortcuts.

Figure: Zoomed Waveform view

 

The Zoomed Waveform view can be brought into view by selecting a phrase, and then pressing a single key Z or selecting Zoom Phrase from the context menu. It overlaps the Content view and is organized in four parts as listed below:

1.      Waveform

2.      Audio Edit toolbar

3.      Navigation and Zoom toolbar

4.      Information bar

 

You can use the following shortcut keys for navigating within or among the bars.

·        To move from one bar to another, press Tab or Shift+Tab.

·        To move among the buttons of a bar, press Left Arrow or Right Arrow.

 

Waveform

This spans the maximum area of the Zoomed Waveform view, and displays the audio waveform of the selected phrase in a size large enough to enable the users to perform the edit operations with ease. It has a phrase label on the top that notifies the position, length, role, etc. of the waveform.

 

The waveform can be further magnified via the Zoom In or Zoom Out buttons of the Navigation and zoom toolbar. You can also use the Zoom Selection button to expand the selected audio in the expanse of the view and get the highly magnified view of the selection. The current zoom level can be preserved and re-applied every time the view is invoked by pressing the button Preserve zoom on this toolbar.

 

Audio Edit toolbar

The Audio Edit toolbar is positioned just below the waveform. It offers the following commands for editing audio. Please note that the keyboard shortcuts for them are the same as that in Content view.

Button

Functionality

Split

Split the phrase at the position of audio cursor.

Merge with Next Phrase

Append the subsequent phrase to the current phrase.

Cut

Cut the selected audio and place it in the application clipboard.

Copy

Copy the selected audio to the application clipboard.

Paste

Paste the waveform from the clipboard to the current position.

Delete

Delete the selected audio.

Phrase Detection

Split the current phrase using the phrase detection algorithm.

Audio Processing (menu)

Opens a menu listing the commands for audio processing like change volume, fade in, etc.

 

Navigation and Zoom toolbar

This toolbar is positioned below the Audio Edit toolbar. It offers the following controls:

Button

Functionality

Close

Close the Zoomed Waveform view and return to the Content view.

Show Next Phrase

Move to the subsequent phrase.

Show Previous Phrase

Move to the phrase just before the current phrase.

Zoom In

Magnify the displayed waveform.

Zoom Out

Shrink the displayed waveform.

Reset

Show the entire waveform of the phrase in the default size.

Zoom Selection

Expand the selected audio in the expanse of the Zoomed Waveform view and get a highly magnified view of the waveform.

Preserve zoom

Preserves the current zoom level to re-apply it every time the view is invoked.

 

Information Bar

Information bar, positioned at the bottommost part of the Zoomed Waveform view, is a thin strip that displays information about the phrase like position, role, etc.

 

This tool is very useful for the non-visual users as it can be used in the similar fashion as the waveform. Keyboard users can move the focus to it and give the navigation, audio editing and selection commands via the usual Content view keyboard shortcuts.

 

Remarks:

·        Zoomed Waveform view opens in the Fine Navigation mode. Therefore, the keyboard shortcuts such as Left Arrow will result in nudging of the audio cursor instead of showing the previous phrase.

·        For the Content view keyboard shortcuts to work properly, the focus should either be on the waveform or the Information bar.
