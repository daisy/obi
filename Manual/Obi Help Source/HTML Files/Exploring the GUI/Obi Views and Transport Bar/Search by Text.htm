Search by Text

 

Although Obi is an audio recording and editing tool, it also contains some non-trivial text. So, a find function is provided to facilitate searching for viewing or editing remote items.

 

You can find an item by clicking Edit->Find in the menu or pressing Ctrl+F. A search bar pops up as shown in the figure below:

 

Figure: Search Bar

 

Type the text you want to search and press Enter.

Obi finds the first item with the label containing the text you typed and shifts the focus to that item. 

 

To find the next occurrence of the text, press F3 or click Next on the search bar. Alternatively, you can select Edit->Find next from the menu.

 

To find the previous occurrence of the text, press Shift+F3 or click Previous on the search bar. Alternatively, you can select Edit->Find previous from the menu.

 

To close the search bar, Click on the Close button.

 

Remarks:

When you execute the find command:

  • If the focus is in the TOC view, Obi will try to match the search text with the section name.
  • If the focus is in the Content view, Obi will try to match the search text with the contents of the visible section.
