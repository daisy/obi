Step by step Installation

 

Before starting the installation, please make sure that you have the administrator privileges. You must also be connected to the Internet for installing the additional tools to make Obi fully functional.

 

Follow the steps given below to install Obi on your system:

1.      Double click on the file named setup.msi in the setup folder and setup wizard opens the welcome page as shown in the figure below:

Figure: Welcome to the Obi X.XSetup Wizard

 

2.      If you do not have .NET framework 2.0 on your system, the installer downloads it through the internet and installs it, before continuing the setup.

3.      Click the  Next button or press Alt+N to move ahead. The wizard shows the license agreement as shown below:

Figure: License Agreement

4.      Read the license agreement thoroughly. Select I Agree radio button, if you agree to the terms and conditions listed in the agreement. If you do not agree to the terms and conditions, the wizard will not continue the installation.

5.      Once you have made the selection, click on the next button or press Alt+N to continue. Setup wizard moves ahead and shows the following :

Figure: Select Installation Folder

6.      In the figure above, the wizard shows the directory where it will store all the files pertaining to the program. By default, it should be C:\program files\the urakawa project\Obi XX. To select a different location, click the browse button next to it or press Alt+R.

7.      If you want the program to be visible to everyone, select the Everyone radio button. Otherwise leave it with the default value: Just Me.

8.      Click on the Next button or press Alt+N. A dialog box appears asking for confirmation of starting installation of Obi on your system:

Figure: Confirm Installation

 

9.      Click on the Next button or press Alt+N to continue installing. The wizard shows the progress bar with a Cancel button, if you want to cancel installation at anytime.

When the installation completes, a dialog box appears giving the status of the installation as shown in the figure below. The wizard will now check the version of DirectX on your computer. If it is earlier than 9C, it will ask for your consent to download DirectX 9C (managed). Install it before finishing.

Figure: Installation Complete

 

10.  Click on the Close Button to close the setup wizard or press Enter.

 

Note: You may go to the previous page of the wizard wherever a Back button is provided. Click on the button or press Alt+B.

You may also cancel the wizard with the Cancel button.

 

After you finish installing Obi, you must install the pipeline backend.

 

Instructions for installing pipeline backend for Obi

 

A light version of the pipeline named Pipeline-lite is used by obi for performing operations like:

·        Encoding an exported DAISY book in mp3 format.

·        Validating a DAISY book.

 

Pipeline-lite needs Java Runtime Environment to be installed. It can be downloaded from JRE Download.

After installing the java runtime environment, follow the steps listed below:

1.     Extract the Pipeline-lite.zip file in installation directory of Obi. By default, it should be C:\program files\the urakawa project\Obi XX.

2.     A folder named Pipeline-lite will be created in Obi's installation directory

3.     In this Pipeline-lite directory, configure file paths in file named pipeline.user.properties. (Normally it works well without any need for configurations. You may have to reconfigure it, if some problems are encountered).
