Sections Menu

 

Provides simple commands to add, insert, rename, or delete a section or a subsection. The keyboard shortcut is Alt+S.

 

Section levels can be changed at any time. You may also merge or split the sections, and they can be marked as Used or Unused.

The figure below illustrates how the Sections menu looks:

 

Figure: Sections Menu

 

The Sections menu contains the following items:

Menu Item

Keyboard Shortcut

Add section

Ctrl+H

Add subsection

Ctrl+Shift+H

Insert section

Ctrl+Alt+H

Rename section

F2

Decrease section level

Ctrl+Shift+Left Arrow

Increase section level

Ctrl+Shift+Right Arrow

Split section

Ctrl+Shift+Q

Merge with next section

Ctrl+Shift+M

Section is used

Ctrl+K
