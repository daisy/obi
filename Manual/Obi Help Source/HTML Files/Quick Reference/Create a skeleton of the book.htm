Create a Skeleton of the Book

 

It is advisable to create a skeleton of the book, before recording the audio content in an Obi project.

 

This process can be subdivided as follows:

1.      Create the sections and subsections in the TOC View.

2.      Create placeholders for the heading of the sections

3.      Create the page phrases wherever they should appear.

 

Creating sections

We have the following commands for creating sections and subsections:

1.      To add a section, click Sections->Add Section on the menu, or press Ctrl+H.

2.      To insert a section, click Sections->Insert Section on the menu, or press Ctrl+Alt+H.

3.      To add a subsection, click Sections->Add subsection on the menu, or press Ctrl+Shift+H.

 

First, add a section to place the contents of the pages at the front of the book, like table of contents, a word by author etc.

Now add empty pages for each page of page-front kind.

Note: You do not need to add a new section, if you want to place all such information in the title section.

 

Now, we will start creating the actual structure of the book. We must identify what should come at the top level and in the subsequent subsections.

 

For example, a novel contains chapters only. As a result, we will have a flat structure in the project, in which each section would be mapped to a chapter. Conversely, in a textbook to be read by students, we may have book-parts, chapters, sections and subsections, nested deeply.

 

The easiest way to create a structure is to do all the work in the TOC View. We will illustrate this by creating a structure for the textbook.

 

We can follow the top down approach or a depth first approach to do it.

 

Depth First Approach

In this approach, we will create the sections and subsections page by page by following the steps given below:

1.      Add a section at the top level for a book-part.

2.      Type in the name of the book-part and press Enter.

3.      Add a subsection for the chapter following the book-part.

4.      Type in the name of the chapter and press Enter.

5.      Add a subsection again for the first topic of the chapter.

6.      Type in the heading of the topic and press Enter.

7.      Add a section for the next topic, or a subsection for the sub-topic. Repeat the process till you are finished with the chapter.

8.      Select the section corresponding to the chapter just finished. Now, add a section for the next chapter and repeat the steps from step 4.

9.      Select the section corresponding to the book-part and repeat the steps again from step 1.

 

Top Down Approach

Create sections for each part of the book first by:

1.      Add a section

2.      Type in the name of the section and press Enter.

3.      Create the next section at the same level and continue the process.

 

Now, make a section for each chapter in a book-part by following the steps below:

1.      Select the section corresponding to the part of the book for which you want to create chapters.

2.      Add a subsection for the first chapter in the book-part.

3.      Type in the name of the chapter, and press Enter.

4.      Add a section below the section just created.

5.      Continue the process from step 3 to create chapters for the same book-part.

6.      Start from step 1 to create chapters for the next book-part.

 

Now, you can create sections for the topics of a chapter in a similar way.

Going a step further, you can create sections for the sub-topics of a topic.

 

Mixing both approaches

You can also mix these approaches as you may want to create the book parts and chapters from the table of contents, following the top-down approach.

Then, you may follow the depth first approach to create sections for the topics and subtopics going from page to page.

 

You may also create the structure in the xhtml format and import it in Obi by creating a new project by importing an xhtml file.

 

Creating heading Phrases

Now, you should create heading phrases for each section, so that the narrator does not miss creating one. Obi uses the first phrase as heading, if not specified explicitly.

 

We will perform this activity in Content View.

Follow the steps given below to create place holders for audio content of a heading of the section:

1.      Go to the first section by pressing Ctrl+Home.

2.      Add a blank phrase by pressing Ctrl+B.

3.      Assign the heading role to the phrase by clicking Phrases->Assign Role->Heading on the menu.

4.      Go to the next section by pressing Down Arrow. Continue the process from step 2 till you reach the end of the list of sections in Content View.

  

Creating pages

Create the page phrases wherever they should appear. If the section spans more than one page, it should contain the page phrases corresponding to each page in the section. Follow the steps given below to create pages:

1.      Go to the top of the list of sections in the Content View by pressing Ctrl+Home.

2.      Go to the section that contains page(s).

3.      Create a phrase, one for each page by pressing Ctrl+B.

4.      Assign a page role to the phrase(s) and specify the page number. It is advisable to check the check box Renumber subsequent pages, so that you do not have to type the number every time.

5.      Continue the above process till you reach the end of the list.

 

The positions of these page phrases can be adjusted after the contents of the section have been recorded, by cutting and pasting them.

 

But, if you want to further facilitate the work of the book narrator, you can insert blank phrases before and after each page phrase for recording the contents of the topic.

For more information on phrases, see Working with Phrases.
