Encoding DTB Audio to MP3

 

You may want to encode the book to a compressed audio format, as WAV files are quite large.

 

Obi allows you to encode the audio files to MP3 by following the steps given below:

1.      Click Tools->Convert audio to MP3.

2.      Select the DAISY standard to which the exported DTB conforms by clicking on the respective radio button in the dialog box just opened.

3.      Click on the OK button. A dialog box opens as shown in the figure below:

 

Figure: Convert audio to MP3

4.      Enter the following fields:

Field

Field Type

Description

Input export directory

Edit Box

Full path of the .opf or .ncc file of DTB.

You can also browse for the file by clicking on the Browse button.

Output directory

Edit Box

Full path of the directory to store the encoded files.

You may also browse for a location by clicking on the Browse button.

Bit rate

Combo Box

Select the bit rate for the MP3 files. Higher the bit rate, higher the quality and hence, more space required.

 

5.      Click on the OK button.

 

Obi will encode the wav files to MP3 and store them in the output directory specified by you. It will also make necessary changes in other files and copy them to the same directory.
