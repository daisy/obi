Em VA-Overwrite as seguintes caixas de verificação estão marcadas:-

Separador áudio:-

Reter silêncio inicial durante a deteção de frases Se marcado, retém silêncio inicial durante a deteção de frases; se desmarcado, elimina esse silêncio.
Usar barra de espaços para parar gravação Se marcado, o Obi pára a gravação; se desmarcado, o comportamento é o padrão, ou seja, apenas faz uma pausa.
Permitir substituição sobre a gravação Se marcado, o Obi abre um espaço na frase para inserir a nova gravação, segundo as regras definidas; se desmarcado, cria uma nova frase a seguir à frase selecionada.
Sempre gravar diretamente desde a barra de controlos Se marcado, é desativada a possibilidade de monitorar o áudio. Como resultado, cada click no botão iniciará a gravação de imediato.
Limitar duração máx. das frases a 50 minutos Se marcado, limita a duração das frases a 50 minutos; se desmarcado, permite frases de qualquer duração.
Mostrar forma de onda ao vivo durante a gravação Se marcado, descreve a forma de onda durante a gravação.
Executar deteção de frases durante a gravação Se marcado, o Obi aplica a deteção de frases enquanto a gravação decorre. No final da gravação, faz as divisões nos pontos detetados.
Renumerar as páginas seguintes quando a gravação terminar Se marcado, o Obi renumera as páginas a seguir ao áudio gravado, logo que a gravação é finalizada.
Juntar as duas primeiras frases detetadas pelo processo de deteção de frases Se marcado, as duas primeiras frases detetadas automaticamente, ficam juntas.
Reprodução rápida sem mudança de tom Se marcado, o Obi preserva o tom ou pitch do áudio, à medida que reproduz a maior velocidade.
Usar botão de gravação para gravar sobre áudio subsequente Se marcado, substitui o áudio gravado pela nova gravação, cada vez que o comando é executado. Neste caso, também deverá estar marcada a caixa sempre gravar a partir da barra de controlos.
Forçar cursor único do áudio Se marcado, o Obi usará apenas um cursor na vez de dois; o cursor vermelho será mapeado para a posição do azul.
Depois de gravar, apagar frases subsequentes Se marcado, o Obi apaga as frases a seguir à que está a ser gravada na secção, logo que a gravação termina.
Não deselecionar ao pressionar stop duas vezes Se marcado, o Obi não desfaz a seleção ao parar de reproduzir; se não marcado, desfaz a seleção.
Preservar páginas ao gravar sobre áudio subsequente Se marcado, as frases de páginas na secção são preservadas.
Usar botão reproduzir e barra de espaços para reproduzir secção Se marcado, o conteúdo é reproduzido por secção.
Desativar criação de novos cabeçalhos e páginas ao gravar Se marcado, os comandos de novo cabeçalho e nova página serão deslocados para o respetivo item, a gravação prossegue, mas nenhum cabeçalho ou página novo é criado.

Separador Projeto:-

Abrir o último projeto Se marcado, o Obi abrirá sempre o projeto da última sessão.
Selecionar marcador quando o projeto é reaberto Se marcado, o Obi abrirá o projeto posicionando o cursor no marcador definido; se não marcado, o projeto será aberto em modo padrão, ou seja, na vista de conteúdo, com o cursor sobre a primeira secção.
Reparar largura da vista de conteúdo Se marcado, o Obi não alterará a orientação de outras frases visíveis, mesmo que o comprimento de uma frase exceda a largura atual da vista de conteúdo. Se não marcado, a largura da vista de conteúdo será alterada consoante o tamanho da frase sob o foco, alterando também o tamanho das outras frases.
Otimizar utilização de memória Se marcado, otimiza o uso de memória por parte do Obi.
Desativar a descartagem para depuração Se marcado, o Obi não permitirá que o utilizador descarte ou elimine o histórico de depuração.
Ativar verificação de espaço em disco Se marcado, pesquisa e informa o utilizador sobre o espaço disponível em disco durante as diversas operações.
Salvar projeto quando a gravação terminar Se marcado, o Obi guarda todas as modificações feitas sempre que a gravação é pausada ou finalizada.
