Overview

 

Audio Format Converter is a stand alone application that alters the file format of the audio files as per user’s requirements. It has simple and user friendly GUI that automatically guides the user through the process.

 

Obi stores its audio data in the WAV format; if user wants to import some pre-recorded audio files in an Obi project, he must make sure that they should be in WAV/RIF format and should comply with the audio preferences of that project.

 

Thus some of the pre-recorded audio files may not be imported in an Obi project due to the following:

·        They are in MP3 format.

·        The audio settings of the WAV files do not match with the audio preferences of the Obi project.

 

In either of the case, Audio Format Converter may be used to make the file format of the pre-recorded audio files compatible with the audio preferences of the Obi project.

Step by step procedure of file conversion

 

  1. Start Audio Format Converter by selecting Programs->Obi->Audio Format Converter from the Start menu.
  2. Place the audio file(s) in the List of input files list box by clicking on the Add button.
  3. Use the Add button again to append more files to the list.
  4. You may delete an audio file from the list by using the Delete button.
  5. Now select the appropriate sampling rate for the audio files in the Sampling rate combo box.
  6. From the Channels combo box, select mono for mono sound or stereo for stereo sound.
  7. Provide the full path of the directory where you want to store the converted files by clicking on the Browse button.
  8. Click on the Start button to start file conversion.

 

Audio Format Converter will alter the file formats of the audio files and store them in the output directory specified in the dialog box.

 

Please note that Obi will import the audio files in the project only if they have the same audio settings as your project. You may check the audio settings of your project as follows:

1.      Open your project in Obi.

2.      Select Preferences from the Tools menu.

3.      Check Sampling rate and Channel from the Audio preferences tab of the preferences dialog just opened.

 

Note:

Audio Format Converter ACCEPTS ONLY MP3/WAV files AS INPUT.

 

Remarks:

  • You may reset the Audio Format Converter dialog to initial state using the Reset button.
  • You may also exit from the Audio Format Converter by clicking on the Cancel button.
